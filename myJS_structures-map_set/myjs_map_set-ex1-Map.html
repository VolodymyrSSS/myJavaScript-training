exercise: використання методів та властивостей Мар

Створіть колекцію по типу ключ/значення на основі структури Мар при цьому використайте різні типи даних для ключів: слово-вирази, числа и булеві.
Також, виведіть в консоль 2 значення зі створеної колекції по ключу та виведіть число елементів в колекції.

Відповідь:

Map – це колекція по типу ключ/значення, як і Object. Але відмінність в тому, що Map дозволяє використовувати ключі будь-якого типу.
Отже визначимо підходячі методи та властивості Map для вирішення цієї задачі з списку та виділимо їх:
new Map() – створює колекцію
map.set(key, value) – записує/встановлює по ключу key значення value
map.get(key) – повертає/отримує значення по ключу чи undefined, якщо ключ key відсутній
map.has(key) – повертає true, якщо ключ key присутній в колекції, а інакще - false
map.delete(key) – видаляє елемент по ключу key
map.clear() – очищає колекцію від усіх елементів
map.size – повертає поточну кількість всіх елементів


let map = new Map();// створюємо колекцію map по типу ключ/значення


Далі в колекції встановимо по ключу значення використавши властивість map.set, при цьому використаємо різні типи значень щоб показати основну відмінність від об'єктів, ось так:

map.set('1', 'str1'); // слово-вираз в якості ключа
map.set(1, 'num1'); // цифра як ключ
map.set(true, 'bool1'); // булеве значення як ключ


Нарешті, виведемо в консоль перші два значення застосувавши властивість map.get при цьому для визначкння наявної кількості елементів використаємо властивість map.size, ось так:

// Ще раз - Map зберігає тип ключів, так що в цьому випадку збережеться 2 різних типи значень:
console.log(map.get(1)); // 'num1'
console.log(map.get('1')); // 'str1'
console.log(map.size); // 3
