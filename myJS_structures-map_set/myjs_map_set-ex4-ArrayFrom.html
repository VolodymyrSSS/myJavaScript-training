myjs-structures: map-set: Перебираемые ключи

Мы хотели бы получить массив ключей map.keys() в переменную и далее работать с ними, например, применить метод .push. Но это не выходит. Почему? Что нужно поправить в коде, чтобы вызов keys.push сработал?

Но это не выходит:
let map = new Map();

map.set('name', 'John');

let keys = map.keys();

// Error: keys.push is not a function
// Ошибка: keys.push -- это не функция
keys.push('more');


Ответ:

Это потому что map.keys() возвращает итерируемый объект, а не массив.
Мы можем конвертировать его в массив с помощью Array.from:


let map = new Map();

map.set('name', 'John');

let keys = Array.from(map.keys());

keys.push('more');

console.log(keys); // name, more
