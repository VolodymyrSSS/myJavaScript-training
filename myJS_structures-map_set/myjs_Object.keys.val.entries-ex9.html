myjs-Object.keys.values.entries: exercise Подсчёт количества свойств объекта

Напишите функцию count(obj), которая возвращает количество свойств объекта.
Постарайтесь сделать код как можно короче. Игнорируйте символьные свойства, подсчитывайте только «обычные».

let user = {
name: 'John',
age: 30
};

console.log( count(user) ); // 2


Решение:

Для решения задачи подходит метод перебора объектов Object.keys(obj) который возвращает массив ключей.
Получив ключи, мы можем использовать свойство для массивов arr.length для получения количество свойств объекта.


function count(obj) {
return Object.keys(obj).length;
}

console.log( count(user) ); // 2
