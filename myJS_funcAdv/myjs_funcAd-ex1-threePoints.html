myjs_funcAdv: exercise Остаточные параметры (...)

На вход функция sumAll() получает аргументы - это числа. Количество аргументов может быть любое. Cоберите все аргументы в массив args и получите сумму из этих чисел.

Пример работы функции
console.log( sumAll(1) ); // 1
console.log( sumAll(1, 2, 3) ); // 6
console.log( sumAll(18, 10, 46, 26) ); // 100


Решение:

Если мы используем для сложения аргументов оператор +, то функция прибавит первых два аргумента. Но если на вход поступило несколько аргументов, функция всё равно посчитает первых два, остальные аргументы не будут учтены, хотя это и не вызовет ошибки.


Например:
function sum(a, b) {
return a + b;
}

sum(1, 2, 3, 4, 5); == 3


Вызывать функцию можно с любым количеством аргументов независимо от того, как она была определена.
Поэтому, для использования всех аргументов и получения их суммы, используем структуру 'остаточные параметры'. Остаточные параметры могут быть обозначены через три точки (...), буквально это значит: «собери оставшиеся параметры и положи их в массив». Далее, мы используем оператор + для получения суммы аргументов-чисел, вот так:


function sumAll(...args) { -здесь args — имя массива
let sum = 0;

for (let arg of args) sum += arg;

return sum;
}

// проверим работу функции:
console.log( sumAll(1) ); // 1
console.log( sumAll(1, 2, 3) ); // 6
console.log( sumAll(18, 10, 46, 26) ); // 100
