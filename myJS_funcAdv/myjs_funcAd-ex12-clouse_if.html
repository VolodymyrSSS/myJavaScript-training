myjs_funcAdv: exercise Замыкание c if

Посмотрите на код. Какой будет результат у вызова на последней строке?

'use strict';
let phrase = "Hello";

if (true) {
let user = "John";

function sayHi() {
console.log(`${phrase}, ${user}`); }
}

sayHi(); // ???


Решение:

ReferenceError: sayHi is not defined - pезультатом будет ошибка.
Функция sayHi объявлена внутри if, так что она живёт только внутри этого блока. Снаружи нет sayHi.


'use strict';
sayHi(); // ReferenceError: sayHi is not defined
