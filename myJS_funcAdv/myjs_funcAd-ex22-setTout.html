myjs_funcAdv: exercise Планирование: setTimeout

Посмотрите на код ниже. Спланируйте вызов sayHi() через 1 секунду.

function sayHi() {
console.log('Salut!');
}


Решение:

Мы можем вызвать функцию не в данный момент, а позже, через заданный интервал времени. Это называется «планирование вызова».
Для этого существуют два метода:
setTimeout позволяет вызвать функцию один раз через определённый интервал времени.
setInterval позволяет вызывать функцию регулярно, повторяя вызов через определённый интервал времени.
Эти методы не являются частью спецификации JavaScript. Но большинство сред выполнения JS-кода имеют внутренний планировщик и предоставляют доступ к этим методам. В частности, они поддерживаются во всех браузерах и Node.js.


Синтаксис setTimeout:


let timerId = setTimeout(func|code, [delay], [arg1], [arg2], ...)


где func|code - Функция или строка кода для выполнения. Обычно это функция. По историческим причинам можно передать и строку кода, но это не рекомендуется;
delay - Задержка перед запуском в миллисекундах (1000 мс = 1 с). Значение по умолчанию – 0.
arg1, arg2… - Аргументы, передаваемые в функцию (не поддерживается в IE9-)


Используем метод setTimeout для решения задачи:


function sayHi() {
console.log('Salut!');
}

setTimeout(sayHi, 1000);


Начинающие разработчики иногда ошибаются, добавляя скобки () после функции -> sayHi()
Это не работает, потому что setTimeout ожидает ссылку на функцию. Здесь sayHi() запускает выполнение функции, и результат выполнения отправляется в setTimeout.

Функция sayRegards() получает на вход два аргумента в виде строки. Спланируйте вызов sayRegards() через 2 секунды.

function sayRegards(phrase, whom) {
console.log( phrase + ' ' + whom );
}


Снова используем синтаксис метода setTimeout для решения задачи указав аргументы функции через запятую:


function sayRegards(phrase, whom) {
console.log( phrase + ' ' + whom );
}

setTimeout(sayRegards, 2000, 'Bonjour à', 'Madame Dubois'); // Bonjour à Madame Dubois


Если первый аргумент является строкой, то JavaScript создаст из неё функцию. Это также будет работать:


setTimeout("console.log('Привiт, друзі')", 3000);


Но использование строк не рекомендуется. Вместо этого используйте функции. Например, так:


setTimeout(() => console.log('ça va?'), 4000);
