myjs_funcAdv_clouse: exercise - cортировать по полю

У нас есть массив объектов (ниже), который нужно отсортировать указав функции поле как здесь: byField(fieldName) ?

массив объектов:
let users = [
{ name: "John", age: 20, surname: "Johnson" },
{ name: "Pete", age: 18, surname: "Peterson" },
{ name: "Ann", age: 19, surname: "Hathaway" }
];


Обычный способ был бы таким:


массив объектов:
// по имени (Ann, John, Pete)
users.sort((a, b) => a.name > b.name ? 1 : -1);

// по возрасту (Pete, Ann, John)
users.sort((a, b) => a.age > b.age ? 1 : -1);


Можем ли мы сделать его короче? То есть, чтобы вместо функции, мы просто писали byField(fieldName).
Напишите функцию byField, которая может быть использована для этого.

скажем, вот так?
users.sort(byField('name'));
users.sort(byField('age'));


Решение:


function byField(field) {
return (a, b) => a[field] > b[field] ? 1 : -1;
}

users.sort(byField('name'));
users.forEach(user => console.log(user.name)); // Ann, John, Pete

users.sort(byField('age'));
users.forEach(user => console.log(user.name)); // Pete, Ann, John
