myjs_funcAdv: exercise Замыкание

Лексическое окружение существует для цикла for, while у каждой итерации. Если переменная объявлена в for(let ...), то она также будет видна только в нём. Что будет выведено в консоле?

for (let i = 0; i < 10; i++) {
// ...какой то код ...
}

console.log(i); // ???


Решение:

Хотя let i визуально находится снаружи {...}. Но конструкция for – особенная в этом смысле,
у каждой итерации цикла своё собственное лексическое окружение с текущим i в нём. И так же, как и в if, ниже цикла i невидима, поэтому будет выведена ошибка.


for (let i = 0; i < 10; i++) {
// ...какой то код ...
// У каждого цикла своё собственное лексическое окружение {i: value}
}

console.log(i); // i is not defined - ошибка, нет такой переменной
