Working with strings

Using methods for strings: str.toUpperCase(), str.split('') and method to convert from array to string arr.join('')

В нас заданий простий рядок слово-виразу. Створіть функцію capitalizeFirstLetter(string), яка буде робити першу букву цього рядка великою.


          // Вихідні дані:

          function capitalizeFirstLetter(string) {
            // ...ваш код ...
          }

          capitalizeFirstLetter('hello'); // 'Hello'


Рішення:


Можливим алгоритмом рішення може бути: спочатку перетворити рядок слово-виразу на масив в якого значеннями будуть кожна буква слово-виразу. Це можна зробити за допомогою метода для рядків split('') та записати цей масив в якусь змінну.
Далі, взяти перший елемент масиву по його індексу (через квадратні дужки) та перевести його у верхній регістр за допомогою метода toUpperCase(). Цим самим ми записали перший елемент масиву що є в дійсності конвертованою першою буквою слово-виразу у верхній регістр.
І вже далі, переводимо масив назад в слово-вираз методом join('') та повертаємо його, ось так: 


          function capitalizeFirstLetter(string) {
            const arrFromString = string.split('');
            arrFromString[0] = arrFromString[0].toUpperCase();
            return arrFromString.join('');
          }

          console.log(capitalizeFirstLetter('hello')); // Hello
