Compare strings no matter of letter order

Working with arrays: operator ===, string method: split(),
arrays methods: sort(), join()


В нас є рядкові слово-вирази які містять в собі одинакові букви і одинакову кіькість, але ці букви розташовані в різному порядку. А є й такі, які мають різну кількість букв. Потрібно створити функцію, яка буде визначити ті рядки, які мають і одинакові букви і одинакову кількість незважаючи на їх порядок.

          // Вихідні дані:

          dog dgo -> true
          dog dfo -> false
          doog ddog -> false
          archer rrahce -> true
          ampersant snerpatma -> true


Рішення:


Якщо ми конвертуємо рядки в елементи масиву, то ми зможемо в подальшому застосувати методи масива. Тобто - посортувати елементи тим самим розмістити елементи в одинаковому порядку. Потім визначити довжину масивів і далі порівняти їх та конвертувати назад в рядки видавши результат true/false.
Отже, для конвертації рядкового слово-виразу в масив, назвемо його isSame, який на вхід отримує два рядкових слово-виразу, застосуємо метод для рядків str.split(), який "розіб'є" рядок слово-виразиу на букви, які є елементами масиву з пробілом як роздільник.
Потім нам їх потрібно відсортувати для того щоб мати упорядковану одинакову послідовність. Це можна зробити методом arr.sort(), ось так:

Ось так: 


          const isSame = (str1, str2) => {
            const letters1 = str1.split('').sort();
            const letters2 = str2.split('').sort();

            console.log(letters1); // 
            console.log(letters2); //
          }


Давайте, подивимось, що ми отримали наразі:


          const isSame = (str1, str2) => {
            const letters1 = str1.split('').sort();
            const letters2 = str2.split('').sort();

            console.log(letters1); // ['d', 'g', 'o']
            console.log(letters2); // ['d', 'g', 'o']
          };

          isSame('dog', 'dgo');


І далі, посортовані елементи масиву контвертуємо методом arr.join() назад в рядковий слово-вираз та строгим порівнянням повертаємо true/false, ось так: 


          const isSame = (str1, str2) => {
            const letters1 = str1.split('').sort();
            const letters2 = str2.split('').sort();
    
            if(letters1.length !== letters2.length) {
              return false;
            }
    
            return letters1.join() === letters2.join();
          }
    
          console.log(isSame('dog', 'dgo')); // true
          console.log(isSame('dog', 'dfo')); // false
          console.log(isSame('doog', 'ddgo')); // false


Тепер, маючи в наявності упорядкованих два масива, ми можемо порівняти їх за довжиною оператора строгого порівняння - === та властивості length. В разі, якщо довжини різні, ми одразу повертаємо false, ось так: 


          if(letters1.length !== letters2.length) {
              return false;
            }


І далі, посортовані елементи масиву контвертуємо методом arr.join() назад в рядковий слово-вираз та строгим порівнянням повертаємо true/false, ось так: 


          const isSame = (str1, str2) => {
            const letters1 = str1.split('').sort();
            const letters2 = str2.split('').sort();
    
            if(letters1.length !== letters2.length) {
              return false;
            }
    
            return letters1.join() === letters2.join();
          }
    
          isSame('dog', 'dgo'); // true
          isSame('dog', 'dfo'); // false
          isSame('doog', 'ddgo'); // false
          isSame('archer', 'rrahce')); // true
          isSame('ampersant', 'snerpatma'); // true
