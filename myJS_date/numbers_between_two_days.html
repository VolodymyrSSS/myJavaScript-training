Working with JS object Date - use the methods getTime() and new Date(dateString)

Обчислення кількості днів між двома датами.
Створіть функцію 'daysBetweenDates' яка буде принймати на вхід дві дати і повертати різницю кількості днів між цими датами. При введенні дат, порядок їх введення не повинен впливати на результат.


Приклад

let dateA = new Date('2020/10/21');
let dateB = new Date('2020/10/29');
daysBetweenDates(dateA, dateB); // Result: 8

або ось так
daysBetweenDates(new Date('2020/10/21'), new Date('2021/10/29'));
// Result: 373


Відповідь:


Щоб вірішити це завдання потрібно спочатку визначити кількість днів яку має найближча дата від початку відліку, потім визначити кількість днів для попередньої дати і вже потім дізнатись різницю (віднявши від найближчої попередню).

Крім того, враховуючи вимогу визначення який з аргументів буде найближчою датою, а який попередньою датою, можемо застосувати функцію Math.abs() при якій порядок аргументів не має значення.

Відлік часу в JavaScript починається з опівночі 1 січня 1970 року. Тому, щоб дізнатись скільки часу пройшло з того часу до визначеної дати, в МІЛІСЕКУНДАХ, існує метод getTime()

А щоб перетворити отримані мілісекунди, наприклад в кількість діб/днів, треба отримані мілісекунди розділити на 1000 * 3600 * 24 де:
1000 - кількість мілісекунд в 1 секунді;
3600 - кількість секунд в 1 годині;
24 - кількість годин в 1 добі;

Також пам'ятаємо, що є 4 способа задати саму дату:
var d = new Date(); - без аргументів, створити об'єкт Date з поточними датою і часом
var d = new Date(milliseconds); - створити об'єкт Date з часом, що дорівнює кількості мілісекунд (тисячна доля секунди), що пройшли з 1 січня 1970 року UTC + 0
var d = new Date(dateString); - якщо аргумент всього один, і це слово-вираз, то з нього ,'прочитується' дата
var d = new Date(year, month, day, hours, minutes, seconds, milliseconds); - створити об'єкт Date з заданими компонентами в місцевому часовому поясі. Обов'язкові тільки перші два аргументи.

Отже, щоб визначити скільки часу пройшло з початку відліку до наприклад поточного дня, треба наприклад зробити так:
var d = new Date();
var n = d.getTime();

Код рішення буде: 

          const daysBetweenDates = (dateA, dateB) => {
            const timeDifference = Math.abs(dateA.getTime() - dateB.getTime())
            
            // Seconds * hours * miliseconds
            return Math.floor(timeDifference / (3600 * 24 * 1000))
          }
