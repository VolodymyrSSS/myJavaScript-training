myjs-date: exercise Покажите день недели

Напишите функцию getWeekDay(date), показывающую день недели в коротком формате: «ПН», «ВТ», «СР», «ЧТ», «ПТ», «СБ», «ВС».

Например
let date = new Date(2012, 0, 3); // 3 января 2012 года
console.log( getWeekDay(date) ); // нужно вывести 'ВТ'


Решение:

Для решения задачи используем метод date.getDay(), который возвращает номер дня недели, начиная с воскресенья. Но если использовать этот метод напрямую, то мы и получим номер дня недели, как здесь:


let date = new Date(2012, 0, 3); // 3 января 2012 года
let day = date.getDay(date);
console.log( day ); // 2


А нам нужно во-первых, создать функцию, во-вторых она должна вывести не номер дня недели, а название дня недели, а именно «ВТ». Поэтому, надо вначале создать массив дней недели, чтобы получить имя нужного дня по его номеру, как здесь:


let days = ['ВС', 'ПН', 'ВТ', 'СР', 'ЧТ', 'ПТ', 'СБ'];


и далее поместить в функцию, которая получает номер дня недели(индекс массива) и выводит значение соответствующего индекса которое соответствует названию дня недели:


function getWeekDay(date) {
let days = ['ВС', 'ПН', 'ВТ', 'СР', 'ЧТ', 'ПТ', 'СБ'];

return days[date.getDay()];
}

let date = new Date(2012, 0, 3);

console.log( getWeekDay(date) ); // 'ВТ'
