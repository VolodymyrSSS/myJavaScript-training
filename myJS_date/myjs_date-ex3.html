myjs-date: exercise День недели в европейской нумерации

В Европейских странах неделя начинается с понедельника (день номер 1), затем идёт вторник (номер 2) и так до воскресенья (номер 7). Напишите функцию getLocalDay(date), которая возвращает «европейский» день недели для даты date.

Например:
let date = new Date(2012, 0, 3); // 3 января 2012 года
console.log( getLocalDay(date) ); // вторник, нужно показать 2


Решение:

Для решения задачи используем метод date.getDay(), который возвращает номер дня недели, но в JS начиная с воскресенья - тобиш c 0.
Но потом необходимо переопределить 0-день, который обозначен как Воскресенье, в 7-день для соответствия европейской нумерации.


function getLocalDay(date) {

let day = date.getDay();

if (day == 0) { // день недели 0 (воскресенье) в европейской нумерации будет 7
day = 7; // переопределили в 7-день
}

return day;
}

let date = new Date(2012, 0, 3);

console.log( getLocalDay(date) ); // 2
