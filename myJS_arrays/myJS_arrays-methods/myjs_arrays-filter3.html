Working with Arrays

using arrays method .filter(), .map(), .slice(), and .join(),
strings method .toUpperCase() and ternary operator cond ? true : false

Напишіть функцію filterRange(arr, a, b), яка отримує масив arr та два числових значення: a та b, які визначають границі діапазону. Пілся ініціалізації функції, здійснюється пошук елементів зі значеннями більшими чи рівними a та меньшими чи рівними b повертаючи масив із знайдених елементів.
Функція повинна повертати новий масив та не змінювати вихідний. Результати виведіть в консоль. 


          // Вихідні дані:

          function filterRange(arr, a, b) {
            // ... ваш код ...
          }

          let arr = [5, 42, 3, 8, 1, 12, 34, 10];

          let filtered = filterRange(arr, 1, 10);

          console.log( filtered ); // 5,3,8,1,10

          console.log( arr ); // [5,42,3,8,1,12,34,10] - має не змінитися


Рішення:


Алгоритмом рішення завдання, може бути застосування методу пошуку елементів в масиві.
Метод arr.find(fn) шукає один елемент (перший який попадеться) на якому функція-колбека поверне true. Але на випадок, якщо знайдених елементів може бути багато, існує метод arr.filter(fn).
Синтаксис цього методу подібний з find(), але filter() повертає масив з усіх підходячих елементів, які відповідають умові пошуку, що зазначені в fn. Отже, використаєм метод filter(fn) де в умові пошуку зазначимо які значення вхідного масиву повинні бути знайдені. Це можна знобити використавши оператори >= або <=. Функція повинна повертати знайдені елелементи, які присвоїмо новому масиву щоб вхідний масив не чіпати, ось так: 


          function filterRange(arr, a, b) {
            return arr.filter((item) => a <= item && item <= b);
          }

          // перевірка роботи функції
          let arr = [5, 42, 3, 8, 1, 12, 34, 10];

          let filtered = filterRange(arr, 1, 10);

          console.log( filtered ); // [5, 3, 8, 1, 10]

          console.log( arr ); // [5, 42, 3, 8, 1, 12, 34, 10] -  не змінився
