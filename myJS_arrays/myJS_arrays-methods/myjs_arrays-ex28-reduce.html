myjs-arrays: arrays_methods: exercises_Получить средний возраст

Напишите функцию getAverageAge(users), которая принимает массив объектов со свойством age и возвращает средний возраст.
Формула вычисления среднего арифметического значения:
(age1 + age2 + ... + ageN) / N.

Например:
let oles = { name: 'Олесь', age: 25 };
let stepan = { name: 'Степан', age: 30 };
let olga = { name: 'Ольга', age: 28 };
let petro = { name: 'Петро', age: 31 };
let lesya = { name: 'Леся', age: 19 };

let arr = [ oles, stepan, olga, petro, lesya,];

console.log( getAverageAge(arr) ) // (25 + 30 + 28 + 31 + 19) / 5 = 26.6


Решение:

Для решения задачи наилучшим образом подходит использование метода reduce(func, initial) который вычисляет одно значение на основе всего массива, вызывая func для каждого элемента и передавая промежуточный результат между вызовами.
Вспомним: let value = arr.reduce(function(previousValue, item, index, array) {...}, [initial]);
Здесь функция применяется по очереди ко всем элементам массива и «переносит» свой результат на следующий вызов.
А previousValue – результат предыдущего вызова этой функции, равен initial при первом вызове (если передан initial). Или можна сказать так - «аккумулирующий» результат предыдущих вызовов функции. По окончании он становится результатом reduce.
Для определения количества студентов будем брать свойство массива length.


>

function getAverageAge(users) {
return users.reduce((prev, user) => prev + user.age, 0) / users.length;
}

// (25 + 30 + 28 + 31 + 19) / 5 = 26.6
console.log( `There are ${arr.length} students. // 5
The avarage age of the group is ${getAverageAge(arr)} years.` ) // 26.6
