Working with Arrays

using arrays method .filter(), property length and operator >

Напишіть функцію findLongWords(words), яка отримує масив рядків (слово-виразів) words та повертає новий масив з елементів, довжина кожного елемента (слова) має мати більше ніж шість символів.
Функція повинна повертати новий масив та не змінювати вихідний. Результати виведіть в консоль. 


          // Вихідні дані:

          function findLongWords(words) {
            // ... ваш код ...
          }

          const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];

          const result = findLongWords(words);

          console.log( result ); // ["exuberant", "destruction", "present"]

          console.log( words ); // ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'] - не міняється


Рішення:


Алгоритмом рішення завдання, може бути застосування методу пошуку елементів в масиві.
Метод arr.find(fn) шукає один елемент (перший який попадеться) на якому функція-колбека поверне true. Але на випадок, якщо знайдених елементів може бути багато, існує метод arr.filter(callbackFn). Метод filter() створює поверхневу копію частини даного масиву, відфільтровану лише до елементів із даного масиву, які пройшли перевірку, реалізовану наданою функцією.
Синтаксис цього методу подібний з find(), але filter() повертає масив з усіх підходячих елементів, які відповідають умові пошуку, що зазначені в колбеці callbackFn, ось:


Його синтаксис:
let results = arr.filter(callbackFn(item, index, array) {
// якщо true - елемент додається до результату, перебор продовжується
// повертається пустий масив у випадку, якщо нічого не знайдено
});


Метод filter() це ітеративний метод. Він викликає надану функцію callbackFn один раз для кожного елемента в масиві та створює новий масив усіх значень, для яких callbackFn повертає правдиве значення. Елементи масиву, які не пройшли перевірку callbackFn, не включаються в новий масив.
Отже, використаєм метод filter(fn) де в умові пошуку зазначимо які значення вхідного масиву повинні бути знайдені. Це можна зробити використавши оператор більше - > та властивість для елементів length. Функція повинна повертати знайдені елелементи, які присвоїмо новому масиву щоб вхідний масив не чіпати, ось так: 


          function findLongWords(words) {
            return words.filter(word => word.length > 6);;
          }

          // перевірка роботи функції
          const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];

          const result = findLongWords(words);

          console.log( result ); // ['exuberant', 'destruction', 'present']

          console.log( words ); // ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'] - не змінився
