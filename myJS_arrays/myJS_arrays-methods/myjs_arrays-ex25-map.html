myjs-arrays: arrays_methods: exercises_Трансформировать в объекты

У вас есть массив объектов user, и у каждого из объектов есть name, surname и id. Напишите код, который создаст ещё один массив объектов с параметрами id и fullName, где fullName – состоит из name и surname. Используйте => функции.

Например:
let oles = { name: 'Олесь', surname: 'Гончар', id: 1 };
let stepan = { name: 'Степан', surname: 'Руданський', id: 2 };
let olga = { name: 'Ольга', surname: 'Кобилянська', id: 3 };

let users = [ oles, stepan, olga ];

let usersMapped = /* ... ваш код ... */

/*
usersMapped = [
{ fullName: 'Олесь Гончар', id: 1 },
{ fullName: 'Степан Руданський', id: 2 },
{ fullName: 'Ольга Кобилянська', id: 3 }
]
*/

console.log( usersMapped[0].id ) // 1
console.log( usersMapped[0].fullName ) // Олесь Гончар


Решение:

На самом деле вам нужно трансформировать один массив объектов в другой. Для решения задачи используем метод map(func) который вызывает функцию для каждого элемента массива и возвращает массив результатов выполнения этой функции.


let oles = { name: 'Олесь', surname: 'Гончар', id: 1 };
let stepan = { name: 'Степан', surname: 'Руданський', id: 2 };
let olga = { name: 'Ольга', surname: 'Кобилянська', id: 3 };

let users = [ oles, stepan, olga ];

let usersMapped = users.map(user => ({
fullName: `${user.name} ${user.surname}`,
id: user.id
}));

/*
usersMapped = [
{ fullName: 'Олесь Гончар', id: 1 },
{ fullName: 'Степан Руданський', id: 2 },
{ fullName: 'Ольга Кобилянська', id: 3 }
]
*/

console.log( usersMapped[0].id ) // 1
console.log( usersMapped[0].fullName ) // Олесь Гончар


Обратите внимание, что для стрелочных функций мы должны использовать дополнительные скобки.
