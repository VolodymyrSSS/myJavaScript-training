myjs-arrays: arrays_methods: join

Покажите на примере возможность использования
метода join?

Ответ:

Метод arr.join(glue) делает в точности противоположное split. Он создаёт строку из элементов arr, вставляя glue между ними..

Например:


let arr = ['Вася', 'Петя', 'Маша'];

let str = arr.join(';'); -объединить массив в строку через;

console.log( str ); // 'Вася;Петя;Маша'
