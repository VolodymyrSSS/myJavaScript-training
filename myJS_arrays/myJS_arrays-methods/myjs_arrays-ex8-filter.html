Покажите на примере возможность использования
метода filter?

Ответ:

Метод find ищет один (первый попавшийся) элемент, на котором функция-колбэк вернёт true. На тот случай, если найденных элементов может быть много, предусмотрен метод arr.filter(fn). Синтаксис этого метода схож с find, но filter возвращает массив из всех подходящих элементов:


Его синтаксис:
let results = arr.filter(function(item, index, array) { // если true - элемент добавляется к результату, и перебор продолжается // возвращается пустой массив в случае, если ничего не найдено });


Например, у нас есть массив пользователей, каждый из которых имеет поля id и name. Давайте возвратим массив из двух первых пользователей и покажем длинну этого массива.

let users = [
{id: 1, name: 'Brian'},
{id: 2, name: 'Peter'},
{id: 3, name: 'Mario'}
{id: 4, name: 'Takada'}
];

let someUsers = users.filter(item => item.id < 3);

console.log(someUsers); // ['Brian', 'Peter']
console.log(someUsers.length); // 2
