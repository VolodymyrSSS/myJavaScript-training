myjs-arrays: arrays_methods: exercises_Фильтрация по диапазону

Напишите функцию filterRange(arr, a, b), которая принимает массив arr, ищет в нём элементы между a и b и отдаёт массив этих элементов.
Функция должна возвращать новый массив и не изменять исходный. Ниже примеры работы функции:

let arr = [5, 3, 8, 1, 18, 2, 11];

let filtered = filterRange(arr, 1, 4);

console.log( filtered ); // 3,1,2 (совпадающие значения)

console.log( arr ); // 5,3,8,1,18,2,11 (без изменений)


Ответ:

Для решения задачи используем метод filter(func) который фильтрует элементы через функцию и отдаёт первое (либо все) значения, при прохождении которых через функцию возвращается true.
Так же для указания границ диапазона используем оператор И(&&)


function filterRange(arr, a, b) {
// добавлены скобки вокруг выражения для улучшения читабельности
return arr.filter(item => (a <= item && item <= b));
}

let arr = [5, 3, 8, 1, 18, 2, 11];

let filtered = filterRange(arr, 1, 4);

console.log( filtered ); // 3,1,2 (совпадающие значения)

console.log( arr ); // 5,3,8,1,18,2,11 (без изменений)
