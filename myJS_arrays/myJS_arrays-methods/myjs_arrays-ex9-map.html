myjs-arrays: arrays_methods: map

Покажите на примере возможность использования
метода map?

Ответ:

Метод arr.map пренадлежит к методам преобразования и упорядочения массива. Он является одним из наиболее полезных и часто используемых. Метод arr.map вызывает функцию для каждого элемента массива и возвращает массив результатов выполнения этой функции.
Или по другому - мы используем его когда нам нужно перебрать массив и вернуть данные для каждого элемента.


Его синтаксис:
let result = arr.map(function(item, index, array) {
// возвращается новое значение вместо элемента
});


Например, здесь мы преобразуем каждый элемент в его длину:


let lengths = ['Bilbo', 'Gandalf', 'Nazgul'].map(item => item.length);

console.log(lengths); // [5,7,6]
