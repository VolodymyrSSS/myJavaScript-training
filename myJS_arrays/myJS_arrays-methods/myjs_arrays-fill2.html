Working with Arrays

using arrays method arr.fill(), and array constructor Array()

Використовуючи конструктор створення масивів, створіть масив arr, який буде мати три елемента пустих об'єктів. Потім заповніть об'єкти парами ключ-значення hi: "hi".
Результати виведіть в консоль. 


          // Вихідні дані:

          отримати масив об'єктів типу: [{ hi: "hi" }, { hi: "hi" }, { hi: "hi" }]


Рішення:


Метод arr.fill() змінює всі елементи в масиві на статичне значення від початкового індексу (за замовчуванням від 0) до кінцевого індексу не включаючи його (за замовчуванням array.length). Він повертає змінений масив.


Його синтаксис:
fill(value)
fill(value, start)
fill(value, start, end)


де value - значення для заповнення масиву, причому зауважте, що всі елементи в масиві будуть мати саме це значення, а якщо value є об’єктом, то кожен слот у масиві буде посилатись на цей об’єкт.

start - індекс від нуля, з якого починається заповнення, це значення перетворюється на ціле число. Від’ємний індекс вираховується від кінця масиву, тобто: якщо start < 0, індекс визначається як start + array.length.
Якщо start < -array.length чи узагалі немає - застосовується 0;
Ну і якщо start >= array.length то індекс не заповнюється узагалі.

end - індекс на якому завершується заповнення, перетворений на ціле число. метод fill() заповнює до кінця, але не включає його. Від’ємний індекс вираховується від кінця масиву, тобто: якщо start < 0, індекс визначається як start + array.length.
Якщо start < -array.length - застосовується 0;
Якщо end >= array.length або end опущено, використовується array.length, що спричиняє заповнення всіх індексів до кінця.
Якщо end розташовано перед або на початку після нормалізації, індекс не заповнюється.
Отже, створюємо спочатку масив з трьох елементів, використовуючи конструктор масивів Array(num), а методом fill({}) заповнюємо масив об'єктами ось так: 


          const arr = Array(3).fill({}); // [{}, {}, {}]


далі, потрібно задати кожному повторяючому об'єкту першу властивість (пара ключ-значення), для цього задаєм ключ-значення першому елементу масиву (що є об'єктом), через квадратні дужки за індексом 0 властивість hi, що буде присвоєно значення "hi". А так, як такими елементами заповняеться увесь масив, то достатньо тільки одному задати властивість і отримаємо її у всіх елементах, які заповнюють масив через метод fill() ось так: 


          const arr = Array(3).fill({}); // [{}, {}, {}]

          arr[0].hi = "hi";

          console.log(arr); // [{ hi: "hi" }, { hi: "hi" }, { hi: "hi" }]
