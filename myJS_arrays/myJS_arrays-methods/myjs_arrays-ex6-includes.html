myjs-arrays: arrays_methods: includes

Покажите на примере возможности использования
специального метода includes?

Ответ:

includes делает по сути то же самое, что и его строковый аналог, но работают с элементами вместо символов. Этот метод используют когда мы хотим проверить наличие элемента, и нет необходимости знать его точный индекс.


Метод arr.includes(item, from) ищет item, начиная с индекса from, и возвращает true, если поиск успешен. Например:


let arr = [1, 56, 0.73, 'tumeric', 0, false, NaN, 107 ];

console.log( arr.includes(56) ); // true
console.log( arr.includes(false) ); // true
console.log( arr.includes(null) ); // false
console.log( arr.includes(NaN) ); // true
console.log( arr.includes('tumeric') ); // true


Этот метод используют строгое сравнение ===, таким образом, если мы ищем false, он находит именно false, а не ноль.
Кроме того, очень незначительным отличием includes является то, что он правильно обрабатывает NaN в отличие от indexOf/lastIndexOf.
