myjs-arrays: arrays_methods: slice

Покажите на примере возможности использования специального метода slice?

Ответ:

Метод arr.slice() возвращает новый массив, в который копирует элементы, начиная с индекса start и до end (не включая end). Оба индекса start и end могут быть отрицательными. В таком случае отсчёт будет осуществляться с конца массива.
Это похоже на строковый метод str.slice, но вместо подстрок возвращает подмассивы.


Его синтаксис:
arr.slice([start], [end])


Например:


let arr = ["t", "e", "s", "t"];

console.log( arr.slice(1, 3) ); // e,s (копирует с 1 до 3, не вкл3)

console.log( arr.slice(-2) ); // s,t (копирует с -2 до конца)


Можно вызвать slice и вообще без аргументов: arr.slice() создаёт копию массива arr. Это часто используют, чтобы создать копию массива для дальнейших преобразований, которые не должны менять исходный массив.


let arr = ['Я', 'изучаю', 'JavaScript', 'прямо', 'сейчас'];

// сделать копию массива 'arr' в массив 'efforts'
let efforts = arr.slice();

console.log( arr ) // теперь ['Я', 'изучаю', 'JavaScript', 'прямо', 'сейчас']
