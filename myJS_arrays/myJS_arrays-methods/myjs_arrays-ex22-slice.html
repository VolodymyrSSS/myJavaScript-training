myjs-arrays: arrays_methods: exercises_Скопировать и отсортировать массив

У нас есть массив строк 'arr'. Напишите код для получения отсортированной копии, но при этом оставить arr неизменённым. Создайте функцию copySorted(arr), которая будет возвращать такую копию.

Например:
let arr = ['HTML', 'JavaScript', 'CSS'];

let sorted = copySorted(arr);

console.log( sorted ); // CSS, HTML, JavaScript
console.log( arr ); // HTML, JavaScript, CSS (без изменений)


Решение:

Для решения задачи вначале применим к массиву 'arr' метод slice(start, end) который создаёт новый массив, копируя в него элементы с позиции start до end (не включая end). Но нам не нужно указывать start и end, а просто использовать метод без указания аргументов, что создаёт копию массива 'arr' для дальнейших преобразований, которые не будут менять исходный массив.
А для сортировки используем сразу метод arr.sort(), который сортирует массив на месте, меняя в нём порядок элементов. Он возвращает отсортированный массив (обычно возвращаемое значение игнорируется).
Объеденим всё это в функции:


function copySorted(arr) {
return arr.slice().sort();
}

let arr = ["HTML", "JavaScript", "CSS"];

let sorted = copySorted(arr);

console.log( sorted ); // ['CSS', 'HTML', 'JavaScript']
console.log( arr ); // ['HTML', 'JavaScript', 'CSS']
