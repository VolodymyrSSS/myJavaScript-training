myjs-arrays: arrays_methods: exercises_Сортировать в порядке по убыванию

У нас есть массив 'arr'. Напишите код для сортировки элементов массива в порядке по убыванию.

Например:
let arr = [5, 3, -5, 8, 1, 18, -10, 2, -2, 11];

console.log( arr ); // [-10, -5, -2, 1, 2, 3, 5, 8, 11, 18]


Решение:

Для решения задачи используем метод arr.sort(), который сортирует массив на месте, меняя в нём порядок элементов. Он возвращает отсортированный массив, но обычно возвращаемое значение игнорируется, так как изменяется сам arr. Или так - по умолчанию элементы сортируются как строки и в действительности выходит, что '2' > '18'.
Чтобы отсортировать массив, нам нужна функция, определяющая порядок, которая знает, как сравнивать его элементы. Кроме того, чтобы использовать наш собственный порядок сортировки, нам нужно предоставить функцию в качестве аргумента arr.sort(). А от функции сравнения требуется любое положительное число, чтобы сказать «больше», и отрицательное число, чтобы сказать «меньше».

// если так задавать, то будет сравнение a b по возростанию
function (a, b) {
return a - b
}

// если так задавать, то будет сравнение a b по убыванию
function (a, b) {
return b - a
}


Используем эту функцию для сравнения пары чисел для сортировки элементов массива. А чтобы сортировка выглядела более аккуратной можем использовать стрелочную функцию:


arr.sort(function(a, b) {
return a-b;
});

// или со стрелочной функцией:
arr.sort((a, b) => b - a);
