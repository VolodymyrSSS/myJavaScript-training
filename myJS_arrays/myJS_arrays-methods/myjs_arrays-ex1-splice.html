myjs-arrays: arrays_methods: splice

Покажите на примерах возможности использования специального метода splice?

Ответ:

Метод arr.splice(str) – это универсальный «швейцарский нож» для работы с массивами. Умеет всё: добавлять, удалять и заменять элементы.


Его синтаксис:
arr.splice(index[, deleteCount, elem1, ..., elemN])


Удаление и замена элементов

Он начинает с позиции index, удаляет deleteCount элементов и вставляет elem1, ..., elemN на их место. Возвращает массив из удалённых элементов. Например:


let arr = ['Я', 'изучаю', 'JavaScript'];

arr.splice(1, 1); -начиная с позиции 1, удалить 1 элемент

console.log( arr ); // осталось ['Я', 'JavaScript']


В примере ниже, мы удалим 3 элемента и заменим их двумя другими.


let arr = ['Я', 'изучаю', 'JavaScript', 'прямо', 'сейчас'];

// удалить 3 первых элемента и заменить их другими
arr.splice(0, 3, 'Давай', 'танцевать');

console.log( arr ) // теперь ['Давай', 'танцевать', 'прямо', 'сейчас']


А в этом примере, видно, что splice возвращает массив из удалённых элементов:


let arr = ['Я', 'изучаю', 'JavaScript', 'прямо', 'сейчас'];

// удалить 2 первых элемента
let removed = arr3.splice(0, 2);

console.log( removed ); // 'Я', 'изучаю' <-- массив из удалённых элементов


Вставка и добавление элементов

Метод splice также может вставлять элементы без удаления, для этого
достаточно установить deleteCount в 0. Например:


let arr = ['Я', 'изучаю', 'JavaScript'];

// с позиции 2 удалить 0 элементов вставить 'сложный', 'язык'
arr.splice(2, 0, "сложный", "язык");

console.log( arr ); // 'Я', 'изучаю', 'сложный', 'язык', 'JavaScript'


В этом и в других методах массива допускается использование отрицательного индекса. Он позволяет начать отсчёт элементов с конца, как в этом примере:


let arr5 = [1, 2, 5];

// начиная с индекса -1 (перед последним элементом) удалить 0 элементов, затем вставить числа 3 и 4
arr.splice(-1, 0, 3, 4);

console.log( arr ); // 1,2,3,4,5
