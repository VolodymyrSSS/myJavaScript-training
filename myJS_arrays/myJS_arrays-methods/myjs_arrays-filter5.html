Working with Arrays

using arrays method arr.filter(fn), arr.includes(), string method str.toLowerCase()

Напишіть функцію filterItems, яка отримує на вхід першим параметром масив рядків (слово-виразів), а другим - підрядок query, та повертає новий масив тільки тих елементів, які включають підрядок query.
По іншому query називають критерій для фільтрації/пошуку. Результати виведіть в консоль. 


          // Вихідні дані:

          const fruits = ["apple", "banana", "grapes", "mango", "orange"];

          function filterItems(arr, query) {
            // .. ваш код ..
          }

          console.log(filterItems(fruits, "ap")); // ['apple', 'grapes']
          console.log(filterItems(fruits, "an")); // ['banana', 'mango', 'orange']


Рішення:


Алгоритмом рішення завдання може бути: фільтрація тих елементів масиву, які містять потрібний підрядок методом arr.filter(fn), в якому визначеємо в функції-колбеку fn критерій для фільтрації. За цим критерієм, спочатку необхідно відшукати ті елементи масиву, які включають заданий підрядок.
Це можна знобити використавши інший метод масивів arr.includes() проте щоб зробити уніфікований пошук необхідно привести елементи до наприклад нижнього регістру за допомогою методу для рядків str.toLowerCase(), ось так: 


          function filterItems(arr, query) {
            return arr.filter((el) =>
              el.toLowerCase().includes(query.toLowerCase())
            );
          }

          console.log(filterItems(fruits, 'ap')); // ['apple', 'grapes']
          console.log(filterItems(fruits, 'an')); // ['banana', 'mango', 'orange']
