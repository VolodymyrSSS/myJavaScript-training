myjs-arrays: arrays_methods: exercises_Фильтрация по диапазону 'на месте'

Напишите функцию filterRangeInPlace(arr, a, b), которая принимает массив arr и удаляет из него все значения кроме тех, которые находятся между a и b. То есть, проверка имеет вид a ≤ arr[i] ≤ b.
Функция должна изменять принимаемый массив и ничего не возвращать. Например:

let arr = [5, 3, 8, 1, 18, 2, 11];

filterRangeInPlace(arr, 1, 4); // удалены числа вне диапазона 1..4

console.log( arr ); // [3, 1, 2]


Ответ:

Для решения задачи должно выполняться условие для елемента массива [i]: если a ≤ arr[i] или arr[i] ≤ b то элемент подходит. Это можно задать оператором ИЛИ (||). Кроме того, надо удалить элемент [i] который не подошол, а это можно осуществить методом arr.splice() где аргументами надо задать элемент [i] который не подошол и число сколько элементов удалить - в нашем случае 1 элемент.

let val = arr[i] -здесь элемент массива [i] присвоили в переменную
if (val < a || val > b) { -здесь сравниваем переменную с нижней и верхней границей
arr.splice(i, 1); -здесь удаляем элемент который не подошол числом 1
i--; -здесь двигаемся по  массиву вниз
}


Для прохода по элементам массива используем цикл for:


for (let i = 0; i < arr.length; i++) { }


Теперь объеденим всё в единый код:


function filterRangeInPlace(arr, a, b) {

for (let i = 0; i < arr.length; i++) {
let val = arr[i];
// удалить, если за пределами интервала
if (val < a || val > b) {
arr.splice(i, 1);
i--;}
}

}

let arr = [5, 3, 8, 1, 18, 2, 11];

filterRangeInPlace(arr, 1, 4); // удалены числа вне диапазона 1..4

console.log( arr ); // [3, 1, 2]
