ARRAYS, arrays_methods: exercises_'на месте'

В нас є масив об'єктів 'cities' із властивостями, де зазначені: в полі 'city' - назва міста, в полі'population' - кількість населення яке проживає в місті та його пригородах.
Необхідно вивести в масив назви міст, в яких населення складає менше ніж 15 млн.


Вихідні дані:

const cities = [
  {city: 'Kyiv', population: 6_000_000},
  {city: 'Tokyo', population: 13_000_000},
  {city: 'New York', population: 22_000_000},
  {city: 'London', population: 17_000_000},
]

// отримати в масиві назви міст, населення яких менше ніж 15 млн
['Kyiv', 'Tokyo'];


Рішення:


Для рішення завдання використоаєм метод map(func) який викликає функцію для кожного елементу масиву і повертає масив результатів виконання цієї функції.
Тобто ми повинні вивести новий масив з іменами міст.
Але перед цим, нам необхідно отримати масив міст, які задовольняють вимоги - щоб кількість населення була менше 15млн.
А для цього, використаємо метод filter(func) який відфільтрує елементи масиву через функцію та віддасть перший чи усі значення, при прохожденні яких через функцію повертається true.
Ось так: 

          const findCities = (population) => {
            cities
              .filter(c => c.population < population)
              .map(c => c.city);
          };

          console.log(findCities(15000000)); // ['Kyiv', 'Tokyo']

Але якщо, для прикладу, нам потрібно задати діпазон (від мінімального до максимального) значень кількості населення, то код може бути змінений ось так:

          const findCities = (popul_min, popul_max) => {
            return cities
              .filter(c => c.population >= popul_min && c.population <= popul_max)
          };
    
          console.log(findCities(10_000_000, 20_000_000)); // ['Tokyo', 'London']
