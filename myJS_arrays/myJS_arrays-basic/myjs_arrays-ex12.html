myjs-arrays: basic_exercises: cумма введённых чисел

Напишите функцию sumInput(), которая:
1. Просит пользователя ввести значения, используя prompt и сохраняет их в массив.
2. Заканчивает запрашивать значения, когда пользователь введёт не числовое значение, пустую строку или нажмёт «Отмена».
3. Выводить каждый раз в консоль массив после введённого числа.
4. Подсчитывает и возвращает сумму элементов массива на экран
P.S. Ноль 0 – считается числом, не останавливайте ввод значений при вводе «0».

Решение:

Внутри функции создаём пустой массив который будет заполнятся введёнными элементами. Ввод значений элементов в массив осуществляем методом push() Запрос значений через цикл while по запросу prompt и сразу вывод полученного массива в консоль. Внутри цикла так же осуществляем проверку на введение нечисла. Обратите внимание на малозаметную, но важную деталь решения. Мы не преобразуем value в число сразу после prompt, потому что после value = +value мы не сможем отличить пустую строку (конец записи) от «0» (разрешённое число). Мы сделаем это позже - когда добавляем в массив.
Внутри функции так же создаём начальное значение суммы для подсчёта введённых значений.


function sumInput(){
let numbers = []; -создаём пустой массив

while (true) {
let value = prompt('Enter the number', 0);

// Проверка на ввод нечисла
if (value === '' || value === null || !isFinite(value)) break;

// тут одновременное добавление в массив и конверсия в число
numbers.push(+value);

console.log(numbers); -выводить каждый раз массив после введения числа
}

let sum = 0; -начальное значение для подсчёта суммы
for (let number of numbers) { -тут цикл for..of для прохода по значениям
sum += number; -тут прибавляем
}
return sum; -возвращаем подсчитаную сумму
}

alert( sumInput() ); - выводим на экран
