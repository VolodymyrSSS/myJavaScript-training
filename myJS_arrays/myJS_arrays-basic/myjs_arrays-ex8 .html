myjs-arrays: basic_exercises: копирование массива

Скопирован ли массив? Что выведет следующий код?

let fruits = ['Яблоки', 'Груша', 'Апельсин'];

let shoppingCart = fruits; - создаём копию
shoppingCart.push('Банан'); - добавляем новое значение в 'копию'

// что в fruits?
console.log( fruits.length ); // ?
console.log( fruits ); // ?


Ответ: 4, массив был скопирован и через копию добавлен новый элемент

Потому, что массивы – это подвид объектов. Обе переменные shoppingCart и fruits являются ссылками на один и тот же массив. Мы можем изменять по любой ссылке этот массив.


console.log( fruits.length ); // 4
console.log( fruits ); // ['Яблоки','Груша','Апельсин','Банан'];
