myjs-arrays: basic_exercises

Мы имеем массив элементов 'fruits'. Выведите в консоль каждый элемент этого массива?

let fruits = ['Яблоко', 'Апельсин', 'Слива'];


Решение:

Для хранения упорядоченных коллекций существует особая структура данных, которая называется массив, Array.
Мы можем получить элемент, указав его номер (индекс) в квадратных скобках. Элементы массива нумеруются, начиная с нуля.


console.log( fruits[0] ); // Яблоко
console.log( fruits[1] ); // Апельсин
console.log( fruits[2] ); // Слива


Замените значение элемента 'Слива' на 'Груша'. Выведите в консоль этот элемент? Потом добавте новый элемент 'Лимон' и выведите в консоль весь массив?

Решение:

Для замены: присваиваем нужному элементу новое значение, а для добавления нового - в квадратных скобках указываем следующий индекс.


fruits[2] = 'Груша'; - перезаписываем значение элемента
console.log( fruits[2] ); // Груша

fruits[3] = 'Лимон';
console.log( fruits ); // ['Яблоко','Апельсин','Груша', 'Лимон']


Выведите в консоль Общее число элементов массива?

Решение:

Общее число элементов массива содержится в его свойстве length:


console.log( fruits.length ); // 4


У нас есть массив 'arr' с разными типами значений?
1. получить элемент с индексом 2 (объект) и затем показать его свойство 'surname', результат выведите в консоль?;
2. получить элемент с индексом 5 (функция) и выполнить её?

let arr = [ 'Папая', 'Гуава', { name: 'Джон', surname: 'Медисон', age: 50 }, true, false, function() { console.log('Здравствуйте!'); } ];


Решение:

В массиве могут храниться элементы любого типа. Для доступа к значениям элементов разных типов, используем обращение к названию массива где в квадратных скобках указываем индекс элемента и далее, через точку - его свойство либо вызываем функцию по правилам её вызова.


// получить элемент с индексом 2 (объект) и затем показать его свойство 'surname'?;
console.log( arr[2].surname ); // Медисон

// получить элемент с индексом 5 (функция) и выполнить её
arr[5](); // Здравствуйте!


Создайте массив 'vegetables' чтоб каждый элемент занимал свою строку. Список свойств массива должен заканчиватся запятой. Для чего запятая?

Ответ:

Список элементов массива, как и список свойств объекта, может оканчиваться запятой. «Висячая запятая» упрощает процесс добавления/удаления элементов, так как все строки становятся идентичными.


let vegetables = [
'Капуста',
'Морковка',
'Свекла', -здесь «Висячая запятая»
];
