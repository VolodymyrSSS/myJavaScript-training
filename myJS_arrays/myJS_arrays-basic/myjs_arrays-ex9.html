myjs-arrays: basic_exercises: операции с массивом используя push/shift/unshift

Произведите 5 операций с массивом:
1. Создайте массив styles с элементами «Джаз» и «Блюз».
2. Добавьте «Рок-н-ролл» в конец.
3. Замените значение в середине на «Классика». Ваш код для поиска значения в середине должен работать для массивов с любой длиной.
4. Удалите первый элемент массива и покажите его.
5. Вставьте «Рэп» и «Регги» в начало массива.

Массив по ходу выполнения операций:
Джаз, Блюз
Джаз, Блюз, Рок-н-ролл
Джаз, Классика, Рок-н-ролл
Классика, Рок-н-ролл
Рэп, Регги, Классика, Рок-н-ролл


Решение:


let styles = ['Джаз', 'Блюз']; -создали массив с 2я элементами

styles.push('Рок-н-ролл'); -добавили в конец Рок-н-ролл

styles[Math.floor((styles.length - 1) / 2)] = 'Классика'; -поменяли значение в середине с учётом изменения длинны

console.log( styles.shift() ); - удалили первый элемент массива 'Джаз' и показали его

styles.unshift('Рэп', 'Регги'); -вставили в начало 'Рэп' и 'Регги'

// посмотрим на созданный массив
console.log( styles ); // ['Рэп', 'Регги', 'Классика', 'Рок-н-ролл']
