myjs-arrays: basic_exercises: многомерный массив

Создайте многомерный массив 'matrix' и выведите в консоль центральный её элемент?

Решение:

Массивы могут содержать элементы, которые тоже являются массивами. Это можно использовать для создания многомерных массивов, например, для хранения матриц. Вот создадим одну:


let matrix = [
[1, 2, 3],
[4, 5, 6],
[7, 8, 9]
];


Далее, выведем центральный её элемент:


console.log( matrix[1][1] ); // 5 - центральный элемент
