myjs-arrays: basic_exercises: реализация массивами метода toString

Как реализуют массивы метод toString?

Ответ: преобразуют в строку где элементы разделены запятыми (без пробелов)

Массивы по-своему реализуют метод toString, который возвращает список элементов, разделённых запятыми. Например:


let arr = [1, 2, 3];

console.log( arr ); // 1,2,3
console.log( String(arr) === '1,2,3' ); // true


А теперь, если попробовать такое:


console.log( [] + 1 ); // '1'
console.log( [1] + 1 ); // '11'
console.log( [1,2] + 1 ); // '1,21'


Массивы реализуют только преобразование toString;
здесь [] становится пустой строкой - ''
[1] становится '1', а [1,2] становится '1,2'.

Когда бинарный оператор плюс '+' добавляет что-либо к строке, он тоже преобразует это в строку, таким образом:


console.log( '' + 1 ); // '1'
console.log( '1' + 1 ); // '11'
console.log( '1,2' + 1 ); // '1,21'
