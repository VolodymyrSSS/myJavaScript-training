myjs-arrays: basic_exercises: вызов функции в контексте массива

Вызов функции в контексте массива. Каков будет результат кода ниже? Почему?

let arr = ['a', 'b'];

arr.push(function() {
console.log( this );
})

arr[2](); // ?


Ответ: вызывается функция которая выводит массив из 3х элементов.

К массиву был добавлен 3й элемент методом push(). И далее, вызов arr[2]() синтаксически – старый добрый obj[method](), в роли obj – arr, а в роли method – 2. Итак, у нас есть вызов функции arr[2] как метода объекта. Соответственно, он получает в качестве this объект arr и выводит массив:


arr[2](); // 'a','b',function


У массива в итоге 3 элемента: сначала их было 2, плюс функция.
