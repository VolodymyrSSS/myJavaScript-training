myjs-arrays: basic_exercises: особенность создания массивов через new Array()

Какая хитрая особенность есть у массива используя другой синтаксис для создания массивов: new Array()?

Ответ:

Как известно, существует ещё один вариант синтаксиса для создания массива: new Array().
Он редко применяется, так как квадратные скобки [] короче но у него есть хитрая особенность. Если new Array вызывается с одним аргументом, который представляет собой число, он создаёт массив без элементов, но с заданной длиной.
Давайте посмотрим, как можно оказать себе медвежью услугу:


let arr = new Array(2); -создастся ли массив [2]?

console.log( arr[0] ); // undefined! -нет элементов, но почему?

console.log( arr.length ); // length 2 - а длинна есть, вот шняга!


Как мы видим, в коде, представленном выше, в new Array(number) все элементы равны undefined. Чтобы избежать появления таких неожиданных ситуаций, мы обычно используем квадратные скобки, если, конечно, не знаем точно, что по какой-то причине нужен именно new Array().
