ARRAYS, arrays_methods: split, sort, join

В нас є слово-вирази які містять в собі одинакові букви, але букви розташовані в різному порядку; а є такі, які мають різні букви. Також в нас є слово-вирази, які містять різну кількість букв. Потрібно визначити ті слововирази, які мають одинакову кількість букв і містять одинакові букви незважаючи на їх порядок.


Вихідні дані:

dog dgo -> true
dog dfo -> false
doog ddog -> false


Рішення:


Для рішення завдання використоаєм метод split() який розіб'є слово-вирази з пробільним розподільником.
Потім нам їх потрібно відсортувати для того щоб мати одинакову послідовність. Це можна зробити методом sort().
І далі вивести строгим порівнянням посортовані букви перевівши їх назад з елемента масива в букви методом join().
Ось так: 

          const isSame = (s1, s2) => {
            const letters1 = s1.split('').sort();
            const letters2 = s2.split('').sort();
    
            if(letters1.length !== letters2.length) {
              return false;
            }
    
            return letters1.join() === letters2.join();
          }
    
          console.log(isSame('dog', 'dgo')); // true
          console.log(isSame('dog', 'dfo')); // false
          console.log(isSame('doog', 'ddgo')); // false
