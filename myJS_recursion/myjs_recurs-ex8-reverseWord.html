          // Вихідні дані:

          const str = "hello"

          function reversing(str) {
            // ... ваш код ...
          }

          console.log( reversing(str) ); // olleh


Рішення:


Можливий алгоритм може бути таким: будувати посимвольно новий рядок символів беручи для нього символ з кінця заданого рядка символів та додавати до нього рядок символів, що залишився. Потім знову брати символ з кінця рядка що залишився, і додавати його до того, що також залишився від попереднього результату. Тобто, вирізати по-одному символ рядоку слово-виразу з кінця та додавати його до наступного символа, який також береться з кінця того самого рядка символів. Така повторяюча дія підпадає під рекурсивний виклик. І так вирізати усі символи з заданого рядка щоб в ньому не залишилось символів. Так буде сформований новий рядок символів але символи у ньому будуть стояти у зворотньому порядку від заданого.
Тому, спочатку визначаємо базу рекурсії, яка буде вважатись такою, коли усі символи були вирізані із заданого рядка, тобто довжина рядка символів буде дорівнювати 0 при цьому функція має повернути пустий рядок, ось так: 


          function reversing(str) {
            if (str.length === 0) return "";
            // піде інший код
          }


Далі йдемо по алгоритму - беремо/вирізаємо останній символ заданого рядку та додаємо до нього рядок символів, що залишився (без того останнього символу що був вирізаний) використавши для цього метод slice(0, str.length-1). Цей "обрізаний" рядок поступає на вхід тої самої функції (рекурсивно). В результаті ми, додамо передостанній символ до останнього та знову рядок, що залишився який поступає на вхід тої самої функції, або рекурсивно викликає сама себе. І так до тих пір, доки з заданого рядка будуть вирізані усі символи. При цьому результат рекурсії також потрібно повертати, тому результуючий код буде ось такий: 


          const str = "hello"

          function reversing(str) {
            // base case
            if (str.length === 0) return "";
            
            // recursion
            return str[str.length - 1] + reversing(str.slice(0, str.length - 1));
          }

          console.log(reversing(str)); // olleh
