Working with strings - use the methods str.startsWith/str.endsWith

У нас є невеликий параграф. Hеобхідно дізнатись чи це той параграф, який нам потрібен, а саме - чи починається він з слово-виразу 'These' та закінчується слово-виразом 'smartphone'? Виведіть результати в консолі.


Приклад

let para = 'These publications developed new ways of laying out and presenting stories, creating the narrative not with slabs of text but with photographs, captions and short pieces of text. For example, Sony this year released the Smartwatch, a two-inch-square screen that can display e-mails, Twitter posts and other pieces of text, all pulled from an Android smartphone.';


Відповідь:


Для вирішення цього необхідно використати методи str.startsWith/str.endsWith Код рішення буде:

          let result = para.startsWith('The'); // true - бо є початком слово-виразу 'These'
          let result2 = para.startsWith('the'); // false - бо чутливий до регістру
          let result3 = para.endsWith('smart'); // false - бо тут береться точно весь слово-вираз 'smartphone.'
          let result4 = para.endsWith('smartphone.'); // true - бо точно такий як  'smartphone.'
          let result5 = para.endsWith('bye'); // false - бо немає такого слово-виразу у 'smartphone.'
