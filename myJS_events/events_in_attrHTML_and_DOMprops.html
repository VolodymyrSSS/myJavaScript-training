Events

Яка функція-виконавець запуститься?
В змінній button є кнопка. Спочатку на ній немає виконавця.
Котрий з виконавців запуститься? Що буде виведено при кліці після виконання коду?


Вихідні дані:

button.addEventListener("click", () => console.log("1"));

button.removeEventListener("click", () => console.log("1"));

button.onclick = () => console.log(2);


Рішення:


У перших двох рядках, виконавець призначається через елементи атрибута. Виконавець в другій функції не буде видалений бо в removeEventListener передається не така сама функція, а інша, хоча з абсолютно одинаковим кодом.
В JavaScript якщо функцію-виконавець окремо не зберегти будь-де, ми не зможемо її і видалити.
Щоб це вирішити потрібно писати код типу такого: 

            function handler() {
                console.log( '1' );
            }
              
            input.addEventListener("click", handler);
            // ....
            input.removeEventListener("click", handler);

Взагалі необхідно сказати, що виконавець завжди зберігається в DOM-об'єкті, а атрибут – лише один зі способів його ініціалізації.
В третьому рядку кода виконавець призначений на властивість DOM-елемента onclick. Призначений виконавець в button.onclick виконається незалежно від addEventListener.
Таким чином, виконається код з першого і третього рядків та буде виведено в консоль "1" і "2".
