Working with context - keyword this, the method сall()

В нас є об'єкт obj1 із своїми властивостями. Також маємо об'єкт obj2 відповідно зі своїми властивостями. В нас написана функція getAnimalSleep, яка використовує ключове слово this та виводить в консоль відповідну інформацію. Використовуючи метод call(), виведіть відповідний контекст для кожного з цих об'єктів у консоль. 

          function getAnimalSleep() {
            console.log(this.animal, "typically sleep between", this.sleepDuration);
          }

          const obj1 = {
            animal: "cats",
            sleepDuration: "12 and 16 hours",
          };

          const obj2 = {
            animal: "dogs",
            sleepDuration: "4 and 7 hours",
          };

          in console.log -> cats typically sleep between 12 and 16 hours
          in console.log -> dogs typically sleep between 4 and 7 hours


Відповідь:


У більшості випадків значення this визначається способом виклику функції і воно вже зв’язане під час виконання функції.
Для того, щоб визначити свій this для кожного об'єкта, потрібно використати метод call(), який ніби прив'яже властивості вказаного об'єкта до ключового слова this цієї функції. Тобто, коли ми викликаємо getAnimalSleep, значення this буде прив’язане до відповідного об’єкта obj, навіть якщо getAnimalSleep не є методом вказаного obj, ось так: 


          function getAnimalSleep() {
            console.log(this.animal, "typically sleep between", this.sleepDuration);
          }

          const obj1 = {
            animal: "cats",
            sleepDuration: "12 and 16 hours",
          };

          const obj2 = {
            animal: "dogs",
            sleepDuration: "4 and 7 hours",
          };

          getAnimalSleep.call(obj1); // cats typically sleep between 12 and 16 hours
          getAnimalSleep.call(obj2); // dogs typically sleep between 4 and 7 hours
