new Promise()

Working with Promise: using method: Promise.then()

В нас є проміс який позитивно вирішується з результатом '123'.
Чи можемо ми одразу отримати результат використовуючи Promise? Як перевірити?

          let promise = new Promise(function(resolve, reject) {
            // задача, яка не потребує часу на вирішення
            resolve(123); // чи видасть одразу результат?
          });


Рішення: так ...


В промісі, функція-виконавець виконується автоматично, при виклику new Promise. Зазвичай, в тілі проміса функція-виконавець здійснює якісь асинхронні дії, і після цього викликає resolve/reject, тобто через якийсь певний час. Але в дійсності, це не обов'язково, 'resolve' чи 'reject' можуть бути викликані одразу, ось так:


          let promise = new Promise(function(resolve, reject) {
            resolve(123);
          });

          console.log(promise); // Promise { fulfilled : 123}


Проте для перевірки, давайте застосуємо метод обробки результатів проміс - .then(), бо на завершених промісах обробники запускаються одразу. Тобто, якщо проміс в стані очікування, обробники в then/catch/finally будуть очікувати його, але якщо проміс вже виконано/завершено, то обробники запускаються одразу, ось так:


          let promise = new Promise(function(resolve, reject) {
            resolve(123); 
          });

          promise.then(console.log); // одразу видасть результат '123'
