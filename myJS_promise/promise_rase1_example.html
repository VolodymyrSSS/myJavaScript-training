Promise.race()

Working with Promise: using method: Promise.race()

В нас є два проміси які позитивно вирішуються відповідними результатами: 'first' та 'second'. До цих промісів ми застосували метод Promise.race().
Який результат ми отримаємо в консолі?

          const p1 = new Promise((res, rej) => {
              setTimeout(res, 500, 'first');
          });

          const p2 = new Promise((res, rej) => {
              setTimeout(res, 100, 'second');
          });

          Promise.race([p1, p2]).then(console.log);


Рішення: 'second'


Коли ми передаємо кілька промісів методу Promise.race, він дозволить той проміс, який буде вирішений першим. У методі setTimeout ми передаємо таймер: 500 мс для першого промісу та 100 мс для другого промісу. Це означає, що p2 дозволиться першим із значенням 'second'. Отже res тепер містить значення 'second', яке виводиться в консоль, ось так:


          Promise.race([p1, p2]).then(console.log); // 'second'
