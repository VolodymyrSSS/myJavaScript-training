Working with strings

methods: str.startsWith(substr) and str.endsWith(substr)

У нас є невеликий параграф. Hеобхідно дізнатись чи це саме той параграф, який нам потрібен, а саме - чи починається він з слово-виразу 'These' та закінчується слово-виразом 'smartphone'? Виведіть результати в консолі. 

          // Вихідні дані:

          let para = 'These publications developed new ways of laying out and
            presenting stories, creating the narrative not with slabs of text but
            with photographs, captions and short pieces of text. For example, Sony
            this year released the Smartwatch, a two-inch-square screen that can
            display e-mails, Twitter posts and other pieces of text, all pulled from
            an Android smartphone.';

          // необхідно отримати підтвердження true/false якщо:
            - починається з підрядка 'The' то отримаєм true
            - починається з підрядка 'the' то отримаєм false
            - закінчується на підрядок 'smart' то отримаєм false
            - закінчується на підрядок 'smartphone.' то отримаєм true
            - закінчується на підрядок 'bye' то отримаєм false


Рішення:


Для вирішення цього завдання потрібно підібрати необхідний метод пошуку підрядка у рядку. Найкращі методи для вирішення підходять str.startsWith(substr) та str.endsWith(substr), ось так:


          para.startsWith('The'); // true - бо є початком слово-виразу 'These'
          para.startsWith('the'); // false - бо чутливий до регістру
          para.endsWith('smart'); // false - бо тут береться точно весь слово-вираз 'smartphone.'
          para.endsWith('smartphone.'); // true - бо точно такий як  'smartphone.'
          para.endsWith('bye'); // false - бо немає такого слово-виразу у 'smartphone.'
