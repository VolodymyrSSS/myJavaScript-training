Advanced functions: exercise setTimeout/setInterval

Що виведеться в консолі першим?

setTimeout( () => console.log("World") );
console.log("Hello");


Рішення: "Hello"

Тут приклад особливого варіанту використання setTimeout з нульовою затримкою:
Синтаксис: setTimeout (func, 0) або просто setTimeout(func).
Застосування планує виклик func настільки швидко, наскільки це можливо. Але надасть право викликати функцію тільки після завершення виконання поточного коду.
Перший рядок ставить виклик в «календар» через 0 мс. Але планувальник перевірить «календар» тільки після того, як поточний код завершиться. Тому "Hello" виводиться першим, а "World" - після нього.


У наведеному нижче коді запланований виклик setTimeout, а потім виконується складне обчислення, для завершення якого потрібно більше 100 мс.
Коли буде виконана запланована функція:
-Після циклу?
-Перед циклом?
-На початку циклу?
Що покаже консоль?


let i = 0; setTimeout(() => console.log(i), 100);
for(let j = 0; j < 100000000; j++) { i++; }


Рішення: Після циклу

Будь-який виклик setTimeout буде виконаний тільки після того, як поточний код завершиться. Останнім значенням i буде: 100000000.
