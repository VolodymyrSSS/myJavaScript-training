Advanced functions: exercise setTimeout/setInterval

Напишіть функцію 'sayHi' щоб вона вивела в консоль привітання 'Ні!!', але тільки через 1 секунду.

Рішення:

Для цього застосуємо метод setTimeout який дозволяє викликати функію один раз через заданий інтервал часу. Синтаксис має вид:
let timerId = setTimeout(func|code, [delay], [arg1], [arg2], ...)
Де func|code - функція або рядок коду для виконання. Зазвичай це функція. З історичних причин можна передати і рядок коду, але це не рекомендується. arg1, arg2... - Аргументи, що передаються в функцію (не підтримуються в IE9-). delay - затримка встановлюється в мілісекундах з розрахунку 1000 мс = 1 секунда. При цьому значення за замовчуванням = 0.

Код для рішення буде мати вид:


                function sayHi() {
                  console.log('Hi!!')  
                }

                setTimeout(sayHi, 1000);


Напишіть функцію 'sayHello', яка на вхід отримує 2 аргумента: 'phrase' де буде сам вираз та 'who' де буде ім'я. і щоб увесь вираз виводився в консоль через 1.5 секунди.

Рішення:

Для цього також застосуємо метод setTimeout, вказавши як аргументи вираз 'Hello' та ім'я 'Julia'.
Код для рішення буде мати вид:


                function sayHello(phrase, who) {
                  console.log( phrase + ', ' + who );
                } 

                setTimeout(sayHello, 1500, 'Hello', 'Julia');
