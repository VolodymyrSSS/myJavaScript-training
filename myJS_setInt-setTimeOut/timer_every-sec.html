Advanced functions: exercise setTimeout/setInterval

Напишіть функцію, яка буде виводити кожної секунди в консоль цифри починаючи від 1 і далі. Тобто, виводиться 1, через 1 секунду 2, через секунду 3 і т. д.

Рішення:


Для рішення завдання використоаєм рекурсивний виклик setTimeout
Ось так: 

          const func = () => {
            //   setTimeout(() => func(), 1000);
            // }
            // func();

Але більш простий метод, через setInterval, ось так:

          const func = () => {
            let i = 1;
            setInterval(() => {
              console.log(i);
              i++;
            }, 1000);
          }

          func();


Ускладнимо завдання. Напишіть функцію, яка буде виводити кожної секунди в консоль цифри від 1 і далі, але вивід наступного числа повинен відбуватись із збільшенням затримки починаючи з 10 мілісекунд? Тобто виводиться число 1, далі виводиться через 20 мілісекунд число 2, далі через 30 мілісекунд число 3 і т. д.

Для рішення завдання використоаєм так само setTimeout
Ось так: 

          const func = () => {
            let a = 1;
            console.log(a);
            a += 1;
            setTimeout(() => func(), 1000 + a * 10);
          }
          func();

Але тут як варіанд можливо забезпечити модульність і використати setTimeout
Ось так: 

          const func = (n = 1) => {
            console.log(n);
            setTimeout(() => func(n + 1), 1000 + n * 10);
          }
          func();
