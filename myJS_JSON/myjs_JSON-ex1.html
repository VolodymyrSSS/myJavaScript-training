myjs-JSON: exercise Преобразуйте объект в JSON, а затем обратно в обычный объект

Преобразуйте user в JSON, затем прочитайте этот JSON в другую переменную.

let user = {
name: 'Modest',
age: 50
};


Решение:

Для для преобразования объектов в JSON используют метод JSON.stringify. Сразу определим другую переменную.


let user2 = JSON.stringify(user);

console.log( user2 );


Что бы обратно преобразовать в обычный объект необходимо использовать метод JSON.parse, вот так:


let user2 = JSON.parse(JSON.stringify(user));

console.log( user2 );
