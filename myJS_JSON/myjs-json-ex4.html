Data Types: exercise JSON

В нас є масив об'єктів 'people'. Виведіть в консоль значення - 35 років Юлі; потім курс 'css' для Івана. Далі, пройдіться по елементам масива та виведіть в консоль усі статуси і потім, на сторінку, усі імена у виді списку. 

                let people = [
                    {
                        name: 'John',
                        age: 50,
                        isAdmin: true,
                        courses: ['html', 'css', 'js']
                    },
                    {
                        name: 'Julia',
                        age: 35,
                        isAdmin: false,
                        courses: ['html', 'css']
                    },
                    {
                        name: 'Mayson',
                        age: 45,
                        isAdmin: false,
                        courses: ['js']
                    }
                ];


Рішення:

JSON (JavaScript Object Notation) - це загальний формат для представлення значень і об'єктів.
Спочатку він був створений для JavaScript, але багато інших мов також мають бібліотеки, які можуть працювати з ним.

JavaScript має такі методи для роботи в цьому форматі:
JSON.stringify(obj) - для перетворення об'єктів в JSON
obj = JSON.parse() - для перетворення JSON назад в об'єкт.

Таким чином, для виведення значення елементу массиву, ми вказуємо індекс цього елементу в квадратних дужках. А для отримання значення властивості об'єкту, ми звертаємось через крапку, отже отримуємо значення: 


                console.log(people[1].age); // 35
                
                console.log(people[0].courses[1]); // css


Для ітерації по елементам масиву, використаємо цикл 'for'. Отже, для отримання усіх статусів, напишемо код: 



                for (let i = 0; i < people.length; i++) {
                    console.log(people[i].isAdmin);
                }

Також, використаємо цикл 'for' для перебору елементів масиву та властивітьс Length. А для виведення імен, використаємо DOM, ось так: 



                // в html запишемо:
                ul id="people" /ul

                // в  script
                let output = '';

                for (let i = 0; i < people.length; i++) {
                    output += 'li>' + people[i].name + '/li>';
                }

                document.getElementById('people').innerHTML = output;
