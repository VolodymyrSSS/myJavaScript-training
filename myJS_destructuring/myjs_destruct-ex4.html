Розумні параметри функції

Існують ситуації, коли функція має багато параметрів. Це особливо
вірно для інтерфейсів користувача. Уявіть функцію, яка створює серію картин.
Вона може мати назву, кількість картин, ширину та висоту картин, вкладені
параметри та багато інших параметрів.
У реальному житті проблема полягає у тому, як запам'ятати порядок усіх аргументів. Інша проблема полягає в тому, як викликати функцію, коли більшість параметрів передавати не треба, і значення за замовчуванням цілком підходять.
Таким чином, в нас є функція яка виводить назву серії картин, їх формат, назви цих картин на екран аукціону (у нас в консоль). Напишіть як функції брати необхідні параметри для їх виводу.
Допустимо, що частину параметрів функція буде брати з об'єкта paintingSeries.


// так дуже погано передавати параметри функції
function showPaintingSeries(seriesName = 'Untitled', width = 95, height = 62, paintingTitles = []) {
  // ...
}

// навіть якщо так:
// тут значення undefined визначаєм там, де потрібно значення по замовчуванню
showPaintingSeries('Nature beauty with buildings', undefined, undefined, ["Roses over the window", "Old trees near the house", "Village on the horizon"]) {   // ...
}
// теж погано, бо виглядає жахливо і код стає нечитаємим, особливо коли
// маємо справу з великою кількістю параметрів

// Отже, розглянемо випадок, коли частину параметрів функція буде отримувати з обєкту:

let paintingSeries = {
  seriesName: "Nature beauty with buildings",
  paintingTitles: ["Roses over the window", "Old trees near the house", "Village on the horizon"]
};


Рішення:

На допомогу приходить деструктуризація! Ми можемо передати параметри як об'єкт,
і функція негайно деструктурує його змінні: Крім того, нагадаємо, що
- для зміни назви властивості використовуєм оператор : ,
- для визначення дефолтного значення використовуєм оператор = .
При чому оператори : та = можуть бути поєднані. Отже деструктуризація цьго об'єкта буде:



function showPaintingSeries({
  seriesName = "Untitled",
  width: w = 95,
  height: h = 62,
  paintingTitles: [painting1, painting2, painting3]
  }) {
    console.log( `${seriesName}, painting format ${w} x ${h}` ); // Nature beauty with buildings, painting format 95 x 62
    console.log( painting1 ); // Roses over the window
    console.log( painting2 ); // Old trees near the house
    console.log( painting3 ); // Village on the horizon
  }

 // Перевірка роботи:
showPaintingSeries(paintingSeries);
