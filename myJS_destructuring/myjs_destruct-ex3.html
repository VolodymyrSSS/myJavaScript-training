Destructuring: Вкладена деструктуризація

В нас є об'єкт "options" який має другий рівень вкладеності у
властивості size (вкладений об'єкт) та у властивості items (вкладений масив).
Покажіть як зробити деструктуризацію цього об'єкта та отримати значення кожної другої властивості вкладеного об'єкту та масиву.
Крім того, додайте властивість title із дефолтним значенням "Menue" та переназвіть властивість "extra" на "otherParameters" із дефолтним значенням "true".


let options = {
  size: {width: 100, height: 200},
  items: ['Cake', 'Donat'],
  extra: true
};

console.log(title); // Menu
console.log(height); // 200
console.log(item2); // Donut


Рішення:

Якщо об'єкт або масив містить інші вкладені об'єкти або масиви, то можна використовувати більш
складні шаблони з лівого боку, щоб отримати вкладені властивості. Власне для деструктуризації
необхідно повторити структуру дерева вкладеностей об'єкта. Крім того, нагадаємо, що
- для зміни назви властивості використовуєм оператор : ,
- для визначення дефолтного значення використовуєм оператор = .
При чому оператори : та = можуть бути поєднані. Отже деструктуризація цьго об'єкта буде:



let options = {
  size: {width: 100, height: 200},
  items: ['Cake', 'Donat'],
  extra: true
}; // зробимо його диструктуризацію як:

let {
  title = "Menue", // визначимо дефолтне значення
  size: {
    width
    height
  },
  items: [
    item1,
    item2
  ],
  extra: otherParameters = true // переназвем властивість із дефолтним значенням
} = options;

 // Перевірка роботи:
console.log(title); // Menue
console.log(height); // 200
console.log(item2); // Donat
console.log(otherParameters); // true
