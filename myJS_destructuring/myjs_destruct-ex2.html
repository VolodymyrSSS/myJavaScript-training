Destructuring: Максимальная зарплата

В нас є об'єкт salaries, у якому визначено заробітну плату програмістів
орієнтовно за 2 тижні. Потрібно створити функцію topSalary, яка буде повертати
ім'я співробітника із найвищою заробітною платою.
Якщо об'єкт salaries пустий, то функція повертає null, а у разі, якщо є декілька
таких високооплачуваних співробітників, можна повернути будь-якого з них.
При вирішенні використайте метод Object.entries та деструктуризацію, для
перебору пар ключ/значення.


let salaries1 = {
   'John': 1100,
   'Pete': 1300,
   'Mary': 1250
   'Samuel': 968
};

або такий обєкт:

let salaries2 = {
   'Matthew': 920,
   'Ketty': 780,
   'Bob': 695
   'Emely': 920
};

або такий:

let salaries3 = {}

function topSalary(salaries) {
   // ... ваш код ...
}

console.log(topSalary(salaries1)); // Pete
console.log(topSalary(salaries2)); // Matthew або Emely
console.log(topSalary(salaries3)); // null - нічого


Рішення:

Отже, функція повинна повертати ім'я і на вхід отримує об'єкт - зарплати.
Властивості/ключі об'єкта ми можемо перебрати методом для об'єктів for..of.
Для необхідності порівняння значень, будемо використовувати оператор < або >.
Крім того, метод Object.entries(obj) – повертає масив із двох значень:
пара [ключ, значення], а тому, тут потрібно зробити деструктуризацію для масивів
щоб можна було брати ці значення масиву для їх подальшого порівняння або перепризначення.



function topSalary(salaries) {

  /* визначаємо змінну для максимальної зарплати, щоб далі її порівнювати і присвоювати туда максимальне
  значення. Також, визначаємо змінну для імені яке буде відповідати максимальному значенню заробітної
  плати із початковим значенням null у разі якщо вхідний об'єкт пустий і нічого не треба повертати: */

  let max = 0;
  let maxName = null;

  for(const [name, salary] of Object.entries(salaries)) {
    if (max < salary) {
      max = salary;
      maxName = name;
    }
  }

  return maxName;
}

 // Перевірка роботи:
console.log(topSalary(salaries3)); // null
