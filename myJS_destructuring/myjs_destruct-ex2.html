myjs-destructuring: exercise Максимальная зарплата

У нас есть объект salaries с зарплатами программистов за 2 недели.
Создайте функцию topSalary(salaries), которая возвращает имя самого высокооплачиваемого сотрудника. Если объект salaries пустой, то нужно вернуть null. Если несколько высокооплачиваемых сотрудников, можно вернуть любого из них.
P.S. Используйте Object.entries и деструктурирование, чтобы перебрать пары ключ/значение.

let salaries = {
'John': 1100,
'Pete': 1300,
'Mary': 1250
'Samuel': 968
};


Решение:

Функция должна возвращать имя, а на вход, должна получать объект с зарплатами ключи которых мы должны перебрать с помощью for of (для объектов). Для сравнения - используем оператор '<>'.
Кроме того, Object.entries(obj) – возвращает массив пар [ключ, значение] поэтому деструктуризацию будем делать как для массивов.


function topSalary(salaries) {

let max = 0;
let maxName = null;

for(const [name, salary] of Object.entries(salaries)) {
if (max < salary) {
max = salary;
maxName = name;}
}

return maxName;
}

console.log(topSalary(salaries)); // Pete
