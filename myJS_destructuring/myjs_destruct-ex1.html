Destructuring: визначення змінних через деструктуризацію об'єкта

В нас є об'єкт 'user'. Потрібно зробити деструктуризацію, яка:
- властивість name визначить у змінній name;
- властивість years визначить у змінній age;
- властивість isAdmin визначить у змінній isAdmin (у разі якщо такої властивості немає - вона має бути визначене із значенням false).


let user = {
   name: 'Amamadoo',
   years: 55
};


Прклад виведення значень в консоль після визначення змінних:



let user = { name: 'Amamadoo', years: 55 };

// ваш код має бути з лівої сторони:
// ... = user

console.log( name ); // Amamadoo
console.log( age ); // 55
console.log( isAdmin ); // false


Рішення:

Для рішення завдання застосуємо деструктуризацію об'єкта при цьому
- для зміни назви властивості використовуєм оператор : ,
- для визначення дефолтного значення використовуєм оператор = .



let user = { name: 'Amamadoo', years: 55 }; // зробимо його диструктуризацію як:

let {name, years: age, isAdmin = false} = user;

 // Перевірка роботи:
console.log( name ); // Amamadoo
console.log( age ); // 55
console.log( isAdmin ); // false
