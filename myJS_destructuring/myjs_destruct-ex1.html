myjs-destructuring: exercise Деструктурирующее присваивание

У нас есть объект 'user'. Напишите деструктурирующее присваивание, которое:
свойство name присвоит в переменную name.
свойство years присвоит в переменную age.
свойство isAdmin присвоит в переменную isAdmin (false, если нет такого свойства)

let user = {
name: 'Amamadoo',
years: 55
};


Пример переменных после вашего присваивания:


let user = { name: 'Amamadoo', years: 55 };

// ваш код должен быть с левой стороны:
// ... = user

console.log( name ); // Amamadoo
console.log( age ); // 55
console.log( isAdmin ); // false


Решение:

Для решения задачи используем обычное деструктурирование:


let user = { name: 'Amamadoo', years: 55 };

// для значения по умолчанию используем =
let {name, years: age, isAdmin = false} = user;l

console.log( name ); // Amamadoo
console.log( age ); // 55
console.log( isAdmin ); // false
