Global scope

Scopes: сутність Global scope

Дайте відповідь, що таке глобальна область видимості?

Відповідь:


Саме слово область видимості - визначає доступність змінних, об’єктів і функцій з різних частин коду.
До ES6 (2015) JavaScript мав лише глобальну область видимості та область видимості функцій. І отже змінні, які оголошені глобально (поза будь-якою функцією), мають глобальну область видимості. Незабуваємо, що в JavaScript об'єкти і функції - також можуть бути змінними, а тому вони також мають глобальну область видимості.
Час життя змінної JavaScript починається з моменту її оголошення. Функціональні (локальні) змінні видаляються, коли функція завершується. А от у веб-браузері глобальні змінні видаляються, коли ви закриваєте вікно (або вкладку) браузера.

Змінні, оголошені за допомогою var, let і const, дуже схожі, коли оголошені поза блоком. Усі вони мають глобальну область видимості: 

          var x = 2;       // оголошена в глобальній області видимості
          let x = 2;        // оголошена в глобальній області видимості
          const x = 2;   // оголошена в глобальній області видимості


Доступ до глобальних змінних можна отримати з будь-якого місця в програмі JavaScript.

          let carName = "Volvo";
          // тут код може використовувати змінну carName

          function myFunction() {
            // код тут також може використовувати carName
          }


У разі, якщо ви присвоїте значення змінній, яка не була оголошена, вона автоматично стане глобальною змінною. Але в "строгому режимі" неоголошені змінні не є автоматично глобальними.
Ось приклад коли змінна carName стане автоматично глобальною, навіть якщо значення присвоєно всередині функції: 

          myFunction();

          // код тут може використовувати змінну carName

          function myFunction() {
	          carName = "Volvo";
          }


Необхідно зазначити, що в JavaScript глобальна область дії – це саме середовище JavaScript. А ось в HTML, глобальною областю є об’єкт глобального вікна Window, і всі глобальні змінні, які визначені за допомогою ключового слова var, належать глобальному об’єкту вікна Window, ось приклад: 

          var carName = "Volvo";
          // код тут може використовувати window.carName


А от глобальні змінні, визначені ключовим словом let, не належать об’єкту Window: 

          let carName = "Volvo";
          // тут код не може використовувати window.carName


ВАЖЛИВО ПАМ'ЯТАТИ:
1) Не створюйте глобальні змінні, хіба що ви цього намірено хочете зробити.
2) Ваші глобальні змінні (або функції) можуть перезаписати змінні (або функції), які вже визначені в об'єкті глобального вікна Window.
3) Будь-яка інша глобальна функція, включаючи глобальний об’єкт вікна Window, може перезаписати ваші глобальні змінні та функції, які ви створили.
