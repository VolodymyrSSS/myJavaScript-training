DOM (data object model)

Working with DOM: using document.body.firstElementChild, document.body.lastElementChild,
або з використанням children типу document.body.firstElementChild,

Для даної сторінки внизу, напишіть код, щоб отримати:
- елемент div;
- елемент ul;
- другий елемент li (з ім'ям Peter);


          // Вихідні дані:

          <html>
          <body>
            <div>Users:</div>
            <ul>
              <li>John</li>
              <li>Peter</li>
            </ul>
          </body>
          </html>


Рішення:


Отже, отримавши DOM-вузел, ми можемо перейти до його найближчих сусідів використовуючи навігаційні посилання.
Є два основних типa навігаційних посилань:

- для усіх вузлів документу: parentNode, childNodes, firstChild, lastChild, previousSibling, nextSibling;
Це добре видно з дерева усіх вузлів документу, ось:

- тільки для вузлів-елементів: parentElement, children, firstElementChild, lastElementChild, previousElementSibling, nextElementSibling;
Це добре видно з дерева усіх вузлів-елементів документу, ось:

Це є тому, що навігаційні властивості які стосуються типу для всіх вузлів, відносяться до всіх типів вузлів в документі. Зокрема, в childNodes знаходяться і текстові вузли, вузли-елементи та вузли-коментарі (якщо вони є). Але в більшості задач, текстові вузли та вузли-коментарі нам не потрібно. І наспрадів, нам більше потрібно маніпулювати в більшості, вузлами-елементами, які є тегами і формують структуру сторінки.

Тому, будемо використовувати в більшості тип вузли-елементи.
Тут, існує два підходи для вирішення поставлених завдань. Для отримання вузла-елемента div: 


          document.body.firstElementChild
          // чи
          document.body.children[0]
          // або використаємо тип "для всіх вузлів" (перший вузел пробіл, тому вибираємо другий)
          document.body.childNodes[1]


Для отримання вузла-елемента ul:


          document.body.lastElementChild
          // чи
          document.body.children[1]


Для отримання другого вузла-елемента li потрібно спочатку визначити вузел-елемент ul, а потім визначити в ньому його останній дочірний елемент li, ось так: 


          document.body.lastElementChild.lastElementChild
