DOM (data object model)

Working with DOM: using classList, methods createElements(), append and method to clone a node elem.cloneNode(true)

Клонування вузлів: cloneNode
У нас на сторінці динамічно створюється div-eлемент із текстом. Як на сторінку вставити таке саме повідомлення? Або як зробити ще таку саму копію? 


          // Динамічно створюваний div-eлемент:

          <script>
            let div = document.createElement('div');
            div.className = "message";
            div.innerHTML = "<strong>Всім привіт!</strong> Ви прочитали важливе повідомлення.";
            document.body.append(div);
          </script>


Результат:




Рішення:


Отже, динамічно ми створили DOM-елемент і вставили його на сторінку у виді повідомлення. Але як вставити на сторінку ще одне таке подібне повідомлення? Звичайно, можна створити функцію і покласти в її тіло код по динамічному створенню повідомлення. Далі викликати функцію створивши копію повідомлення. Також можна склонувати існуючий div та просто замінити текст всередини нього.
Інколи, коли в нас є великий елемент, це може бути скоріше та простіше. Але виклик elem.cloneNode(true) - створює глибокий клон елементу - із усіма атрибутами та дочірними елементами. А якщо ми викличемо elem.cloneNode(false) - тоді клон буде без дочірних елементів.
Тому, давайте створимо копію повідомлення, ось так: 


          <style>
            .message {
              padding: 15px;
              border: 1px solid #d6e9c6;
              border-radius: 4px;
              color: #3c763d;
              background-color: #dff0d8;
            }
          </style>

          <script>
            let div = document.createElement('div');
            div.className = "message";
            div.innerHTML = "<strong>Всім привіт!</strong> Ви прочитали важливе повідомлення.";
            document.body.append(div);

            let div2 = div.cloneNode(true); // клонувати повідомлення
            div2.querySelector('strong').innerHTML = 'Всім бувайте!'; // змінити клонований елемент в частині тега 'strong'

            div.after(div2); // показати клонований елемент після існуючого div
          </script>
