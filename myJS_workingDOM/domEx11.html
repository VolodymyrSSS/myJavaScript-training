Рішення:


Перш за все, нам потрібно знайти усі зовнішні посилання, а потім задати стиль як вказано в завданні через link.style.color = 'orange';.
А тому, шукаємо усі посилання використовуючи універсальний метод пошуку вузол-елемента по його тегу: elem.querySelectorAll(css) який повертає усі елементи які є всередині elem, які відповідають даному CSS-селектору. Цей метод є досить ефективний бо можна використовувати будь-який CSS-селектор.
Також, потрібно присвоїти результат пошуку змінній, яку будемо використовувати для роботи з властивостями, ось так: 


          let links = document.querySelectorAll('a');
          console.log(links); // NodeList(7) [a, a, a, a, a, a, a]


Як ми бачимо, ми отримали усі посилання - колекцію з семи елементів довжиною 7. Ну тепер можна пройтись циклом for...of по елементам (по кожному посиланню) та задамо кожному елементу стиль як вказано в завданні. Так ми поки задамо усім елементам оранжевий колір, ось так:


          let links = document.querySelectorAll('a');

          for (let link of links) {
            link.style.color = 'orange';
          }


А тепер ми можемо відфільтрувати тільки ті посилання, які є зовнішніми посиланнями (які відповідають умовам завдання). Для цього, визначаємо для кожного посилання a обов'язковий атрибут href по якому і будемо фільтрувати і можемо це зробити за допомогою метода elem.getAttribute(name), що дозволить отримати значення атрибуту. Звичайно, ми могли взяти це значення використавши властивість DOM-елемента через крапку, типу link.href, але нам потрібно отримати значення, яке записано точно як в HTML. Значення DOM-властивості може бути іншим, наприклад, DOM-властивість href – завжди містить повний URL, навіть якщо атрибут містить відносний URL чи просто #hash, а нам потрібно отримати оригіналье значення. Ось приклад: 


          <a id="a" href="#hello">link</a>
          <script>
            // атрибут
            console.log(a.getAttribute('href')); // #hello

            // DOM-властивість
            console.log(a.href ); // повний URL типу http://site.com/page#hello
          </script>


Тому беремо це значення використавши саме метод elem.getAttribute(name), ось так: 


          let links = document.querySelectorAll('a');

          for (let link of links) {
            let href = link.getAttribute('href'); // властивість DOM-об'єкту

            link.style.color = 'orange';
          }


Коли браузер завантажує сторінку, він читає (також говорять: "парсить") HTML і генерує з нього DOM-об'єкти. Для вузлів-елементів більшість стандартних HTML-атрибутів автоматично стають властивостями DOM-обєктів. Маючи DOM-об'єкт, ми можемо здійснювати фільтрацію використовуючи JavaScript методи. А тому здійснюєм фільтрацію використовуючи оператор if та для продовження ітерації, оператор continue, ось так: 


          let links = document.querySelectorAll('a');

          for (let link of links) {
            let href = link.getAttribute('href'); // властивість DOM-об'єкту

            if (!href) continue; // немає атрибута

            if (!href.includes('://')) continue; // немає протокола

            if (href.startsWith('http://internal.com')) continue; // внутрішнє посилання

            link.style.color = 'orange';
          }


Можна розглянути, більш простий шлях – додати одразу перевірку в CSS-селектор.
Для перевірки використовуємо CSS-селектор [attribute*="value"] який використовується для вибору елементів, що мають специфічний атрибут, всі значення яких включають значення "://". Також, використовуємо CSS-селектор [attribute^="value"] який використовується для виключення тих елементів, що мають специфічний атрибут, всі значення яких починаються саме із "http://internal.com".
Далі, використати вже перевірений селектор в універсальному методі отримання властивостей DOM-елементів, по яких проходимся циклом та задаєм кожній властивості відповідний стиль, ось так: 


          let selector = 'a[href*="://"]:not([href^="http://internal.com"])';
          let links = document.querySelectorAll(selector); // NodeList(3) [a, a, a]

          links.forEach(link => link.style.color = 'orange');
