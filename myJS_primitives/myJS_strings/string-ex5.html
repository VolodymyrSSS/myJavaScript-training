Working with strings str.slice(pos)

Виділити число: є вартість у вигляді слово-виразу '$120'. Тобто спочатку йде знак валюти, а потім - число. Створіть функцію extractCurrencyValue(str), яка буде з такого слово-виразу виділяти числове значення і повертати його? 

          console.log( extractCurrencyValue('$120') === 120 ); // true


Відповідь:


Значення слово-виразу в JavaScript не можна змінити. Тому треба взяти тільки частину слово-виразу без першого символу за допомогою методу str.slice(start [, end]), який повертає частину взятого слово-виразу від start до, не включаючи, end. В нашому випадку візьмемо частину від першого символу (не нульового) і до кінця. А щоб перевести слово-вираз в число, використовуємо унарний +. Ось так: 

          function extractCurrencyValue(str) {

            return +str.slice(1);

          }

          
          // перевірка роботи функції:

          console.log( extractCurrencyValue('$120') ); // 120
