Проверка на спам: напишите функцию checkSpam(str), возвращающую true, если str содержит 'viagra' или 'XXX', а иначе false.
Функция должна быть нечувствительна к регистру. Работа функции проверяется так:

checkSpam('buy ViAgRA now') == true
checkSpam('free xxxxx') == true
heckSpam('innocent rabbit') == false


Решение:

Первое что надо сделать, это перевести строку в нижний регистр, что даст нам возможность осуществлять поиск без учёта регистра. А потом, использовать метод str.includes() для поиска необходимой подстроки в результате которого выводится либо true (значит есть подстрока) либо false (подстроки нету) при этом, будем использовать оператор || (ИЛИ) для определения нужных подстрок.


Переводим строку в нижний регистр:
let str = 'buy ViAgRA now';
let lowerStr = str.toLowerCase();
console.log( lowerStr );


А теперь, давайте посмотрим если в строке есть искомая подстрока используя метод str.includes(); и если есть должно быть выведено в консоле true, вот так:


console.log( lowerStr.includes('viagra') ); // true


Далее, завернём всё в функцию checkSpam(str). Эта функция должна возвращать работу метода str.includes() где получим или true или false.


function checkSpam(str) {
let lowerStr = str.toLowerCase();

return lowerStr.includes('viagra') || lowerStr.includes('xxx');
}

// проверка работы функции:
console.log( checkSpam('buy ViAgRA now') ); // true
console.log( checkSpam('free xxxxx') ); // true
console.log( checkSpam('innocent rabbit') ); // false
