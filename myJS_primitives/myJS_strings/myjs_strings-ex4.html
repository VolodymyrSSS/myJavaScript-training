myJS-primitives: exercises strings

Выделить число: есть стоимость в виде строки '$120'. То есть сначала идёт знак валюты, а затем – число. Создайте функцию extractCurrencyValue(str), которая будет из такой строки выделять числовое значение и возвращать его
Например:

console.log( extractCurrencyValue('$120') === 120 ); // true

Решение:

Содержимое строки в JavaScript нельзя изменить. Поэтому надо взять подстроку, отбросив первый символ, методом str.slice(start [, end]) (возвращает часть строки от start до, не включая, end). А чтобы перевести подстроку в число, используем унарный +.


function extractCurrencyValue(str) {
return +str.slice(1);
}

// проверка работы функции:
console.log( extractCurrencyValue('$120') ); // 120
