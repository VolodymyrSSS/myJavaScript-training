Working with strings str.length, str.slice(start [, end])

Cтворіть функцію truncate(str, maxlength), яка перевіряє довжину слово-виразу str і, якщо вона перебільшує maxlength, замінює закінчення str на '…' так, що її довжина стає дорівнювати maxlength. Результатом функції повинен бути або обрізаний слово-вираз, або той самий, якщо обрізання не потрібно. 

          truncate('Ось, що мені хотілось би сказати на цю тему:'', 20); // Ось, що мені хотіло…
          

          truncate('Всім привіт!', 20); // Всім привіт!


Відповідь:


Для вирішення задачі потрібно, щоб функція порівнювала довжину слово-виразу str із maxlength. Якщо довжина більша, то маємо обрізати та додати спеціальний Юнікодний символ '…'. І для цього потрібно визначити як правельно взяти довжину слова з урахуванням maxlength щоб її порівняти.
Щоб визначити будь-яку довжину, використаємо числову властивість str.length. Але в умовах задачі дано також і maxlength, тому будемо брати її для порівняння. Потрібно врахувати і те, що в кінець потрібно додати спеціальний Юнікодний символ '…', тобто довжину при цьому потрібно визначати як maxlength - 1 щоб вистачило місця на додавання трьох крапок (доречі, це не те ж саме, що такі ... - три крапки).
Таким чином, щоб порівнювати довжину слово-виразу str.length з maxlength - 1 використаємо виконання умови через тернарний оператор ? :, в якому якщо порівняння не задовільняє умову, то беремо тільки частину слово-виразу за допомогою методу str.slice(start [, end]) і в кінець додаєм '…', а якщо задовільняє - то повертаємо слово-вираз цілком. Ось так: 


          function truncate(str, maxlength) {
            return (str.length > maxlength ? str.slice(0, maxlength - 1) +  '…' : str);
          }

          // перевірка роботи функції
          let string = 'Ось, що мені хотілось би сказати на цю тему:';
          console.log(truncate(string, 20)); // Ось, що мені хотіло…
          string = 'Всім привіт!';
          console.log(truncate(string, 20)); // Всім привіт!
