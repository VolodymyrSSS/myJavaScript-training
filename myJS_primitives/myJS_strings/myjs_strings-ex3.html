myJS-primitives: exercises strings

Усечение строки: создайте функцию truncate(str, maxlength), которая проверяет длину строки str и, если она превосходит maxlength, заменяет конец str на '…', так, чтобы её длина стала равна maxlength.
Результатом функции должна быть та же строка, если усечение не требуется, либо, если необходимо, усечённая строка.
Например:

truncate('Вот, что мне хотелось бы сказать на эту тему:'', 20) = 'Вот, что мне хотело…'

truncate('Всем привет!', 20) = 'Всем привет!'


Решение:

Функция должна возвращать проверку на длинну строки.
Строка, которую мы возвращаем, должна быть не длиннее maxlength, поэтому, если мы обрезаем строку, то мы должны убрать на один символ больше, чем maxlength — чтобы хватило места на многоточие. В качестве многоточия здесь используется … — ровно один специальный юникодный символ (это не то же самое, что ... — три точки).
А для проверки длинны строки используем тернарный оператор где условием ставим сравнение с заданной максимальной длинной, первым результатом определяем обрезанную строку (подстрока) полученую методом str.slice(start [, end]) (возвращает часть строки от start до, не включая, end) + один специальный юникодный символ ('…'), а другим результатом - подстроку целиком так как удовлетворяет условие по длине.


function truncate(str, maxlength) {
return (str.length > maxlength) ? str.slice(0, maxlength - 1) + '…' : str;
}

// проверка работы функции:
console.log( truncate("Вот, что мне хотелось бы сказать на эту тему:", 20) ); // Вот, что мне хотело…
console.log( truncate("Всем привет!", 20) ); // Всем привет!
