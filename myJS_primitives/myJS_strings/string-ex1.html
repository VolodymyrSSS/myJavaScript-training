Working with strings str[pos], str.charAt(pos), for..of

Створіть змінну 'str' і присвойте їй значення 'abcde'. Звертаючись до окремих символів слово-виразу, виведіть в консоль символ 'а', символ 'b' та символ 'е'. Використайте інший спосіб виведення символів, та виведіть ним символ 'с' і символ 'd'. В чому різниця між методами? 

          let str = 'abcde';


Відповідь:


В JavaScript отримати символ, який займає позицію pos, можна за допомогою квадратних дужок: [pos]. Вивидемо в консолі символи використавши квадратні дужки:

          console.log(str[0]); //виведе символ 'a'
          console.log(str[1]); //виведе символ 'b'
          console.log(str[4]); //виведе символ 'e'


Крім того, для отримання символа слово-виразу можна використати інший метод str.charAt(pos). Застосуємо цей метод та вивидемо символи 'с' і 'd':

          console.log(str.charAt(2)); //виведе символ 'c'
          console.log(str.charAt(3)); //виведе символ 'd'


Але, різниця між ними така: якщо символ з позицією [pos] відсутній тоді метод поверене, undefined, а charAt — порожній рядок, ось:

          console.log(str[5]); //виведе undefined
          console.log(str.charAt(6)); //нічого не виведе або порожній рядок

Створіть змінну 'str' і присвойте їй значення 'abcde'. 'Пройдіться' по слово-виразу 'str' та виведіть кожен його символ в окремому рядку. 

          a,b,c,d,e - вивести окремо кожен символ в рядку


Відповідь:


Якщо потрібно 'перебрати' посимвольно будь-який слововираз, використовують for..of, ось так: 

          let str = 'abcde';

          for (let char of str) {
            console.log(char); // a,b,c,d,e (char —  'a', потiм 'b', потiм 'c' i т. д.)
          }
