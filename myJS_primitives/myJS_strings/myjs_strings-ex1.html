myJS-primitives: exercises strings

Напишите функцию ucFirst(str), возвращающую строку str с заглавным первым символом. Например:

ucFirst('margo') == 'Margo';


Решение:

Содержимое строки в JavaScript нельзя изменить. Нельзя взять символ посередине и заменить его. Как только строка создана — она такая навсегда. Поєтому, нужно пересоздать строку на основе существующей, с заглавным первым символом методом str.toUpperCase(), остальные буквы (или подстроку) добавим с помощью метода str.slice(start [, end]) при этом укажем только один аргумент - начало откуда брать (со второго символа) и автоматически до конца.


let newStr = str[0].toUpperCase() + str.slice(1);


Однако есть небольшая проблемка. Если строка пуста, str[0] вернёт undefined, а у undefined нет метода toUpperCase(), поэтому мы получим ошибку.
Для решения этого мы используем для взятия первого символа не квадратные скобки, а метод str.charAt(0), поскольку он всегда возвращает строку (для пустой строки — пустую), вот так:


let str = 'margo';
let newStr = str.charAt(0).toUpperCase() + str.slice(1);
console.log( newStr ); // Margo


Но если всё же использовать квадратные скобки, надо сделать проверку на пустую строку, а для этого используем цикл if. Завернём всё в функцию ucFirst(str) по условию задачи, вот так:


function ucFirst(str) {
if (!str) { -здесь если пустая строка
return str; - то вернёт её
} else {
return str[0].toUpperCase() + str.slice(1); }
}


И теперь проверим работу функции:


let str = 'walter';
console.log( ucFirst(str) ); // Walter

let str = '';
console.log( ucFirst(str) ); // ничего (пустая строка)

let str = 'sue';
console.log( ucFirst(str) ); // Sue
