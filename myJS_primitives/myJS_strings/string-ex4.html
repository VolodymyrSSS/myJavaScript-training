Working with strings str.toLowerCase(), str.includes('substring')

Перевірка на спам: напишіть функцію checkSpam(str), що повертає true, якщо str містить 'viagra' або 'XXX', а інакше false.
Функція повинна бути нечутлива до регістру. 

          checkSpam('buy ViAgRA now') == true

          checkSpam('free xxxxx') == true

          checkSpam('innocent rabbit') == false


Відповідь:


Перше що треба зробити, це перевести рядок в нижній регістр, що дасть нам можливість здійснювати пошук без урахування регістру. Ось так: 

          let str = 'buy ViAgRA now';

          let lowerStr = str.toLowerCase();

          console.log( lowerStr );


А потім, використовувати метод str.includes() для пошуку необхідної частини слово-виразу в результаті якого виводиться або true (значить є частана) або false (частини немає), ось так:

          console.log( lowerStr.includes('viagra') ); // true


При цьому, будемо використовувати оператор || (АБО) для отримання умови пошуку ще однієї частин варіанту слово-виразу. Ось так:

          console.log( lowerStr.includes('viagra') || lowerStr.includes('xxx') ); // true


Далі, огорнемо все у функцію checkSpam(str). Ця функція повинна повертати роботу методу str.includes() де отримаємо або true або false, ось так:

          function checkSpam(str) {

            let lowerStr = str.toLowerCase();

            return lowerStr.includes('viagra') || lowerStr.includes('xxx');

        }

        

        // перевірим роботу функції:

        console.log( checkSpam('buy ViAgRA now') ); // true

        console.log( checkSpam('free xxxxx') ); // true

        console.log( checkSpam('innocent rabbit') ); // false
