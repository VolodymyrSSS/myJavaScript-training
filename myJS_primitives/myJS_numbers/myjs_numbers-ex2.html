myJS-primitives: exercises методы у примитивов

Создайте скрипт, который запрашивает ввод (через prompt) двух чисел 'a' и 'b', а после показывает их сумму.
P.S. Есть «подводный камень» при работе с типами.

Решение:

Ввод будем осуществлять через prompt. Прибавление чисел осуществляем оператором +.
Для преобразования строки в число используем унарный оператор + перед prompt, который преобразует значение в числовой формат.
В противном случае, a и b будут строками, и после суммирования произойдёт конкатенация двух строк, а именно: '1' + '2' = '12'.


let a = +prompt('Enter the first number', '');
let b = +prompt('Enter the second number', '');

alert( a + b );
