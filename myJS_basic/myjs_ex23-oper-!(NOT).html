Basic JS: логический оператор '!'

Оператор 'НЕ' представлен восклицательным знаком ! (по русски - 'НЕ')
Синтаксис довольно прост:



result = !value;


Оператор принимает один аргумент и выполняет следующие действия:
Сначала приводит аргумент к логическому типу true/false.
Затем возвращает противоположное значение.


Например:
console.log( !true ); // false
console.log( !0 ); // true
console.log( !null ); // true
console.log( !1 ); // false
console.log( !-24 ); //false
console.log( !2347 ); // false


Большое применение логического оператора '!' это его двойное применение.
Двойное 'НЕ' - используют для преобразования значений к логическому типу:


Например:
console.log( !!0 ); // false
console.log( !!1 ); // true
console.log( !!"non-empty string" ); // true
console.log( !!null ); // false


Приоритет 'НЕ' является наивысшим из всех логических операторов, поэтому он всегда выполняется первым, перед && или ||.

Напишите условие if для проверки, что значение переменной age НЕ находится в диапазоне 14 и 90 включительно.
Задайте переменную чтобы она отвечала диапазону и вывелось 'Так как ваш возраст такой-то вам вход бесплатный'.
Используйте для вывода в консоль шаблонные строки. Напишите два варианта: первый с использованием оператора !, второй – без этого оператора.

Решение с оператором !:


let age1;
age1 = 10;
if( !(age1 >= 14 && age1 <= 90 )) {
console.log(`Because your age is ${age1}, entrance free.`);
};


Решение без оператора !:


let age;
age = 95;
if( age < 14 && age > 90 ) {
console.log(`Because your age is ${age}, entrance free.`);
};
