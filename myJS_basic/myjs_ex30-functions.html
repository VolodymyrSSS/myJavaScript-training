Решение используя операторы '?', или '||'


function showMessage(age) {

Напишите функцию min(a,b), которая возвращает меньшее из чисел a и b.
Пример вызовов: min(2, 5) == 2; min(3, -1) == -1; min(1, 1) == 1;
Решите эту задачу с вариантом используя оператор '?'

Решение:



function min(a, b) {
if(a < b) {
return a;
} else {
return b; }
};
// проверим работу функции:
console.log(min(2, 5)); // 2
console.log(min(3, -1)); // -1
console.log(min(1, 1)); // 1


В случае равенства a == b не имеет значения, что возвращать.

Решение используя оператор '?'



function min(a, b) {
return a < b ? a : b;
}
