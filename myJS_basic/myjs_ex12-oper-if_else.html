Basic JS: использование конструкции if..else

Используя конструкцию if..else, напишите код, который будет спрашивать: „Какое «официальное» название JavaScript?“ Если пользователь вводит «ECMAScript», то показать: «Верно!», в противном случае – отобразить на экране: «Не знаете? ECMAScript!»

Решение:



let value = prompt("Какое 'официальное' название JavaScript?", "");
if (value == 'ECMAScript') {
alert('Верно!');
} else {
alert('Не знаете? ECMAScript!');
}


Используя конструкцию if..else, напишите код, который получает число через prompt, а затем выводит в alert:
1, если значение больше нуля,
-1, если значение меньше нуля,
0, если значение равно нулю.
Предполагается, что пользователь вводит только числа.

Решение:


let value = prompt("Enter the number?", 0);
if (value > 0) {
alert (1);
} else if (value < 0) {
alert (-1);
} else {
alert (0);
}
