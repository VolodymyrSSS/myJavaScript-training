Basic JS: условные операторы: if..else

1. Создайте страницу, которая спрашивает 'В каком году появилась спецификация ECMAScript-2015?'
Если был введён 2015 год то вывести 'Да вы знаток!', если был введён другой год - то вывести 'А вот и неправильно!'
2. Создайте такую же страницу, которая на этот же вопрос выводит ответы в зависимости от введённого года: если был введён год до 2015, то ответ 'Это слишком рано...'; если был введён год после 2015, то вывести 'Это поздновато', ну а если 2015 то вывести 'Абсолютно верно!'

Все выводы делайте в консоль. Там где это возможно используйте шаблонные строки.

Решение1:



let year = prompt("What year did the ECMAScript-2015 specification appear?", "2014");
if (year == 2015){
alert("Yes you are a connoisseur!");
} else { alert("And here it is wrong!");
}



Решение2:

Для этого используем блок 'else if'



let year = prompt("What year did the ECMAScript-2015 specification appear?", "2016");
if (year < 2015){
alert("This is too early...");
} else if (year > 2015) {
alert("It's too late");
} else {
alert("Absolutely right");
}


Блоков else if может быть и больше. Присутствие блока else не является обязательным.
