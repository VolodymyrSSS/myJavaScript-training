Working with increment ++ and decrement --

В нас є змінна n та три стрілочні функції які збільшують значення перемінної оператором increment ++, причому перша функція збільшує змінну в 'префіксній' формі, а друга в 'постфіксній' формі, ну і третя функція просто повертає значення n.
Що буде виведено у консолі в результаті такого збільшення?

          let n = 0;

          const fn1 = () => ++n;
          const fn2 = () => n++;
          const fn3 = () => n;

          console.log(fn1()); // ???
          console.log(fn2()); // ???
          console.log(fn3()); // ???


Рішення: 1, 1 та 2


Перше, що видно, змінній n було присвоїно значення '0'. В момент виклику першої функції f1 значення n було спочатку збільшено на одиницю, а вже потім було повернуто і в консолі виводиться '1'.
Натомість, в функції f2, значення спочатку повертається і виводиться в консоль, а вже потім n збільшується на одиницю, тобто в консоль було виведено '1', а n стало рівне '2'.
Ну і остання функція просто повертає '2', ось так:


          let n = 0;

          const fn1 = () => ++n;
          const fn2 = () => n++;
          const fn3 = () => n;

          console.log(fn1()); // 1
          console.log(fn2()); // 1
          console.log(fn3()); // 2
