Basic JS: инкремент++/декремент--

1. Чему будут равны переменные a, b, c и d в примере ниже?
2. Объясните детально на примерах какая разница между инкремент и декремент ?


let a = 1, b = 1;
let c = ++a;
let d = b++;


Ответ1:


let a = 1, b = 1;
let c = ++a;
let d = b++;

console.log( a ); // 2, значение увеличено один раз
console.log( b ); // 2, значение увеличено один раз
console.log( c ); // 2
console.log( d ); // 1
так как:
console.log( ++a ); // 2, префиксная форма возвращает новое значение
console.log( b++ ); // 1, постфиксная форма возвращает старое значение


Ответ2:

Одной из наиболее частых операций, как и во многих других языках программирования, является увеличение или уменьшение переменной на единицу.
Для этого в JavaScript существуют специальные операторы:


Инкремент ++ увеличивает на 1:



let counter = 2;
counter ++; // работает как counter = counter + 1, просто запись короче
cosole.log( counter ); // 3


Декремент -- уменьшает на 1:



let counter = 2;
counter--; // работает как counter = counter - 1, просто запись короче
cosole.log( counter ); // 1


Инкремент/декремент можно применить только к переменной. Попытка использовать его на значении, типа 5++, приведёт к ошибке.
Операторы ++ и -- могут быть расположены не только после, но и до переменной.
Когда оператор идёт после переменной – это «постфиксная форма»: counter++. «Префиксная форма» – это когда оператор идёт перед переменной: ++counter

Как мы знаем, все операторы возвращают значение. Операторы инкремент/декремент не исключение. Префиксная форма возвращает новое значение, в то время как постфиксная форма возвращает старое (до увеличения/уменьшения числа).
Можно разяснить по другому: префиксная форма вначале увеличивает значение велечины а потом присваивает это значение, а постфиксная форма вначале присваивает значение велечины а потом увеличивает его.



let counter = 1;
let a = ++counter; // Здесь префиксная форма увеличения counter, она возвращает новое значение 2
cosole.log(a); // 2


Теперь посмотрим на постфиксную форму - меняем ++counter на counter++



let counter = 1; let a = counter++; // Здесь постфиксная форма counter++ также увеличивает counter, но возвращает старое значение (которое было до увеличения). cosole.log(a); // 1


Подведём итоги:


Если результат оператора не используется, а нужно только увеличить/уменьшить переменную – без разницы, какую форму использовать:


let counter = 0; counter++; ++counter; console.log( counter ); // 2, обе строки сделали одно и то же


Если хочется тут же использовать результат, то нужна префиксная форма:



let counter = 0;
console.log( ++counter ); // 1


Если нужно увеличить и при этом получить значение переменной до увеличения – постфиксная форма:



let counter = 0;
console.log( counter++ ); // 0


Операторы ++/-- могут также использоваться внутри выражений. Их приоритет выше, чем у арифметических операций. Например:



let counter = 1;
console.log( 2 * ++counter ); // 4


Сравните с:



let counter = 1; console.log( 2 * counter++ ); // 2, потому что counter++ возвращает "старое" значение


Хотя технически всё в порядке, такая запись обычно делает код менее читабельным. Лучше использовать стиль «одна строка – одно действие»:



let counter = 1;
console.log( 2 * counter );
counter++;
