Basic JS: інкремент++/дикремент--

1. Чому будуть дорівнювати значення a, b, c та d в прикладі нижче?
2. Поясніть детально на прикладах, яка різниця між інкрементом та дeкрементом?


Які будуть значення?
let a = 1, b = 1;
let c = ++a;
let d = b++;


Відповідь на 1е:


console.log( a ); // 2, значення збільшено один раз
console.log( b ); // 2, значення збільшено один раз
console.log( c ); // 2, значення збільшено і одразу результат присвоєно змінній
console.log( d ); // 1, значення збільшено але присвоєно попереднє значення змінної
через те, що:
console.log( ++a ); // 2, префіксная форма повертає нове значення
console.log( b++ ); // 1, постфіксная форма повертає старе значення


Відповідь на 2е:

Однією з найчастіших операцій, як і в багатьох інших мовах програмування, є збільшення або зменшення змінної на одиницю.
Для цього у JavaScript існують спеціальні оператори:


Інкремент ++ збільшує значення на 1:



let counter = 2;
counter++; // працює як counter = counter + 1, але запис коротший
cosole.log( counter ); // -> 3


Декремент -- зменшує значення на 1:



let counter = 2;
counter--; // працює як counter = counter - 1, але запис коротший
cosole.log( counter ); // -> 1


Інкремент/декремент можна застосувати лише до змінної. Спроба використовувати його на значенні, типу 5++, призведе до помилки.
Оператори ++ і -- можуть ставитись не тільки після, а й до змінної.
Коли оператор йде після змінної – це постфіксна форма: counter++.
Префіксная форма – це коли оператор ставиться попереду змінної, ось як тут: ++counter.

Як відомо, всі оператори повертають значення. Оператори інкремент/декремент не є винятком.
Префіксна форма повертає одразу нове значення, тоді як постфіксна форма повертає старе (до збільшення/зменшення числа).
Можна пояснити інакше: префіксна форма спочатку збільшує значення величини на одиницю і потім надає це значення. А постфіксна форма спочатку привласнює попереднє значення величини, а потім його збільшує на одиницю.



let counter = 1;
let a = ++counter; // Тут префіксная форма збільшує counter на одиницю і повертає новее значення -> 2
cosole.log(a); // -> 2


Тепер подивимось на постфіксну форму - міняемо ++counter на counter++



let counter = 1; let b = counter++; // Тут постфіксна форма counter++ також збільшує величину counter на одиницю, проте повертає старе значення (яке було до збільшення). cosole.log(b); // -> 1


Таким чином:


Якщо результат оператора не використовується, а потрібно лише збільшити/зменшити змінну – не має значення, яку форму використовувати:


let counter = 0;
counter++;
++counter;
console.log( counter ); // -> 2,
В обох випадках було збільшено значення counter на одиницю, і в результаті отримали 2


Якщо одразу потрібно взяти результат, то застосовують префіксну форму:



let counter = 0;
console.log( ++counter ); // -> 1


А в разі, коли потрібно збільшити на одиницю та при цьому мати значення змінної до збільшення, використовують – постфіксну форму:



let counter = 0;
console.log( counter++ ); // -> 0


Оператори ++/-- можуть використовуватися всередині виразів. Їхній пріоритет вище, ніж в арифметичних операціях, наприклад:



let counter = 1;
console.log( 2 * ++counter ); // -> 4


І тепер порівняйте з:



let counter = 1;
console.log( 2 * counter++ ); // -> 2, тому, що counter++ повертає попереднє значення до збільшення - тобто число '1' і має приорітет над операцією множення.
