Основы JS: работа с переменными

1. Объявите две переменные: admin и name.
2. Запишите в name строку 'Василий'.
3. Скопируйте значение из name в admin.
4. Выведите в консоль admin (должно вывести «Василий»).
5. Проверте являются ли такими же обычными символами, как буквы: знак доллар '$' и знак подчеркивания '_' ?;

Решение 1-4:

Каждая строчка решения соответствует одному шагу задачи:



var admin, name; //две переменных через запятую
name='Василий'; // присвоили значение 'Василий'
admin=name; // скопировали значение в другую переменную
console.log(admin); // 'Василий'


Решение 5:

С этой целью объявим переменную с именем '$' и переменную с именем '_' и сделаем простое вычесление:


var $ = 1; //объявили переменную с именем '$'
var _ = 2; // переменная с именем '_'
console.log( $ + _ ); // 3
