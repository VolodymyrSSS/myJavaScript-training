Подключение script в HTML-документ

1. Создайте страницу, которая отобразит в консоле сообщение «Я JavaScript1!».
2. Покажите на экране второе сообщение «Я JavaScript2!» подключив внешний скрипт.

Решение1:

Программы на JavaScript могут быть вставлены в любое место HTML-документа с помощью тега-script.
Как правило, только простейшие скрипты помещаются в HTML. Более сложные выделяются в отдельные файлы.
Для подключения нескольких скриптов надо использовать несколько тегов, примеры:
script src="/js/script1.js" /script
script src="/js/script2.js" /script

Используя тег-script вставим наш код в HTML-документ при этом результат работы скрипта выведем в консоль.


console.log( "Я JavaScript1!" );


Решение2:

Если есть много JavaScript-кода, его желательно поместить в отдельный файл.
Файл скрипта можно подключить к HTML с помощью атрибута src:

Польза от отдельных файлов в том, что браузер загрузит скрипт отдельно и сможет хранить его в кеше.
Другие страницы, которые подключают тот же скрипт, смогут брать его из кеша вместо повторной загрузки из сети. И таким образом файл будет загружаться с сервера только один раз. Это сокращает расход трафика и ускоряет загрузку страниц.

Для второго сообщения укажем относительный путь от текущей страницы. Здесь надо учесть, что к примеру запись script src="alert.js" /script
будет означать, что файл "alert.js" находится в текущей папке; а запись
script src="./alert.js" /script находится в папке что лежит выше по иерархии.


script src="alert.js" /script - относительный путь к внешнему скрипту
