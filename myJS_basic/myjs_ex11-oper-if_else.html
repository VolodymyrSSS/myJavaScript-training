Basic JS: оператор 'if'

1. Выполнится ли код при таком условии ?


if (0) { alert( 'Привет' ); }


Ответ1: нет не выполнится - 0 is falsy

Оператор if (…) вычисляет выражение в скобках и преобразует результат к логическому типу и, если результат true, то выполняет блок кода, если false - то нет.
Далее, по правилам преобразования типов: число 0, пустая строка "", null, undefined и NaN становятся false. Из-за этого их называют «ложными» («falsy») значениями. Остальные значения становятся true, поэтому их называют «правдивыми» («truthy»).

2. А выполнится ли код при таком условии ?


if ('0') { alert( 'Привет' ); }


Ответ2: выполнится - любая строка, кроме пустой (а строка '0' – не пустая), в логическом контексте становится true

3. А при таком ?


if (1) { alert( 'Привет' ); }


Ответ3: выполнится всегда так как здесь число 1 - truthy
