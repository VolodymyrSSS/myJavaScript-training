Basic JS: условные операторы: if..else и '?'

1. Назначте переменную в зависимости от условия;
2. Спросите сколько лет;
3. если нет 18 - допуск отменить и вывести на экране false;
4. Eсли больше 18 - допуск разрешить и вывести на экране true
Решите операцию двумя способами: используя if..else, и более коротким и простым способом - используя тернарный оператор.

Решение c if..else:



let accessAllowed;
let age = prompt('How old are you?', '18');

if (age < 18) {
accessAllowed = false;
} else {
accessAllowed = true;
}

alert( accessAllowed );


Решение c тернарным оператором:

Так называемый «условный» оператор «вопросительный знак» позволяет нам сделать это более коротким и простым способом.
Оператор представлен знаком вопроса ?. Его также называют «тернарный», так как этот оператор, единственный в своём роде, имеет три аргумента.

Синтаксис: let result = условие ? значение1 : значение2;
Сначала вычисляется условие: если оно истинно, тогда возвращается значение1, в противном случае – значение2.



let age = prompt('How old are you?', '18');
let accessAllowed = (age < 18) ? false : true;
alert( accessAllowed );
