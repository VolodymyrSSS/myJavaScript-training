Работа с переменными: использование тернарного оператора в условиях более чем одного условия

1. Спросите сколько лет;
2. Назначте переменную в зависимости от сообщения которое выводится;
3. если меньше 5 лет - вывести на экране 'Привет пацанёнок!';
4. если меньше 18 лет - вывести на экране 'Привет дружище!';
5. если меньше 75 лет - вывести на экране 'Здравствуйте, сер';
6. если до 100 и более лет - вывести на экране 'Здравствуйте, какой почтенный возраст';
Решите операцию двумя способами: вначале используя тернарный оператор, потом используя оператор if..else.

Решение для '?':

Последовательность операторов вопросительного знака ? (или 'тернарного оператора') позволяет вернуть значение, которое зависит от более чем одного условия.


let age = prompt('Your age?', 18);
let message = (age < 5) ? 'Hi, kid!' :
(age < 18) ? 'Hello my friend!' :
(age < 75) ? 'Hello, Sir!' :
'Hello, what a respectable age!';

alert( message );



Поначалу может быть сложно понять, что происходит. Но при ближайшем рассмотрении мы видим, что это обычная последовательная проверка:
Первый знак вопроса проверяет age < 5.
Если верно – возвращает 'Здравствуй, малыш!'. В противном случае, проверяет выражение после двоеточия „":"“, вычисляет age < 18.
Если это верно – возвращает 'Привет дружище!'. В противном случае, проверяет выражение после следующего двоеточия „":"“, вычисляет age < 75. Если это верно – возвращает 'Здравствуйте, сер'. В противном случае, возвращает выражение после последнего двоеточия – 'Какой почтенный возраст!'.


Смысл оператора вопросительный знак ? – вернуть то или иное значение, в зависимости от условия. Пожалуйста, используйте его именно для этого. Когда вам нужно выполнить разные ветви кода – используйте if.


Решение для if..else:

При чтении глаза сканируют код по вертикали. Блоки кода, занимающие несколько строк, воспринимаются гораздо легче, чем длинный горизонтальный набор инструкций.


let message;
let age = prompt('Your age?', 18);
if (age < 5) {
message = 'Hi, kid!';
} else if (age < 18) {
message = 'Hello my friend!';
} else if (age < 75) {
message = 'Hello, Sir!';
} else {
message = 'Hello, what a respectable age!';
}
alert( message );
