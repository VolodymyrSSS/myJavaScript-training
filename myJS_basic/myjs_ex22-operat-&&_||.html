Basic JS: логические операторы && и ||

Что выведет код ниже?


console.log( null || 2 && 3 || 4 );


Ответ:


Выведет 3 так как приоритет оператора && выше, чем ||, поэтому он выполнится первым.
Результат 2 && 3 = 3, поэтому выражение приобретает вид: null || 3 || 4
И теперь для оператора 'ИЛИ' результатом является первое истинное значение - 3.

Что выведет код ниже?


console.log( console.log(1) && console.log(2) );


Ответ:


Сначала 1, затем undefined.
Вызов console.log не возвращает значения, или, иначе говоря, возвращает undefined.
Первый оператор && выполнит первый console.log(1).
Получит undefined и на нём же и закончит так как найдено ложное значение.
Поэтому до правого console.log(2) дело не дойдёт, вычисления закончатся на левом.

Какие из перечисленных ниже console.log выполнятся?


if (-1 || 0) console.log( 'first' );
if (-1 && 0) console.log( 'second' );
if (null || -1 && 1) console.log( 'third' );


Ответ:

1 и 3 выполнятся, 2 - нет



if (-1 || 0) alert( 'first' ); // Выполнится
// Результат -1 || 0 = -1, в логическом контексте true

if (-1 && 0) alert( 'second' ); // Не выполнится
// Результат -1 && 0 = 0, в логическом контексте false

if (null || -1 && 1) alert( 'third' ); // Выполнится
// оператор && имеет больший приоритет, чем || так что -1 && 1 выполнится раньше // Результат: null || -1 && 1 -> null || 1 -> 1
