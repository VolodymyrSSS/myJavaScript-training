convertion methods to/from obj Object.fromEntries(array) and Object.entries(obj),
iterations over data structures for plain objects as for...of, using arrays method map()

У нас є об'єкт prices з відповідними властивостями. Необхідно отримати подібний об'єкт у якого будуть ті самі властивості, але подвоєні значення.
Результати виведіть в консоль. 


          // Вихідні дані:

          let prices = {
            banana: 1,
            orange: 2,
            meat: 4,
          };

          let doublePrices = // ... ваш код

          // перевірка
          console.log( doublePrices.meat ); // 8


Рішення:


Алгоритмом рішення завдання, може бути таким: перевести об'єкт в масив, що дасть змогу використати методи масивів. Виконати потрібні дії над елементами масиву та конвертувати масив назад у об'єкт.
Отже, переводимо об'єкт в масив за допомогою метода Object.entries(obj), в результаті якого отримаємо масив в якому кожним елементом, позначимо як entry, є також масив з двох елементів - ключ об'єкта та його значення.
Далі, застосувавши метод масивів map(), зможемо отримати перший елемент масиву як entry[0], та подвоїти друге значення елементу масиву як entry[1] * 2, наразі, це буде виглядати ось так: 


          Object.entries(prices).map(entry => [entry[0], entry[1] * 2])


Необхідно зазначити, що потрібно отримати також масив з двух елементів, щоб можна було конвертувати масив масивів назад в об'єкт де перший елемент масиву буде ключом, а другий елемент, який подвоїли, буде його значенням. Конвертацію назад в об'єкт будемо робити методомObject.fromEntries(array) куда замість array треба помістити увесь попередній вираз. Це і буде, власне, рішенням, ось: 


          let prices = {
            banana: 1,
            orange: 2,
            meat: 4,
          };

          let doublePrices = Object.fromEntries(
             Object.entries(prices).map(entry => [entry[0], entry[1] * 2])
          );

          console.log(doublePrices.meat); // 8
