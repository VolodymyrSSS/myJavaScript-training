Рішення:


Для вирішення цього завдання потрібно просто в функції повернути загальну кількість полів заданого об'єкту. Для цього, використаєм властивість масивів arr.length, а щоб можна було взяти значення length, потрібно отримати цей масив та його елементи - тобто поля об'єкту отримати як елементи масиву, або перевести об'єкт в масив. Це можна зробити за допомогою методу ітерації по полям об'єкту Object.keys(obj) (повертає масив ключів об'єкту). Отже, найкоротший код рішення буде: 


          function countProps(userN) {
            return Object.keys(userN).length;
          }
