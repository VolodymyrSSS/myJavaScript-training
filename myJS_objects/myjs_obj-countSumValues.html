Working with Objects

object method to iterate through the values Object.values(obj),
iterations over data structures for plain objects as for...of, arrays method reduce()

У нас є об'єкт salaries у якому (умовно) зазначені у властивостях їх заробітні плати за місяць. Напишіть функцію sumSalaries(salaries), яка підрахує загальну суму заробітку усіх працівників. Покажіть щонайменше два можливих рішення.
Результати виведіть в консоль. 


          // Вихідні дані:

          let salaries = {
            John: 1952,
            Pete: 3020,
            Mary: 2487,
          };

          function sumSalaries(salaries) {
            // ... ваш код
          }

          // перевірка
          console.log( sumSalaries(salaries) ); // 7459


Рішення1:


Алгоритмом рішення завдання, може бути таким: створюємо змінну, якій буде присвоєно результат обчислення, звичайно він на початку буде дорівнювати нулю. Далі потрібно пройтись по всім властивостям об'єкту за допомогою циклу for...of та взяти суму усіх їх значень використавши метод Object.values(obj), ну і звичайно, повертаємо значення підрахованої суми, ось так: 


          function sumSalaries(salaries) {
            let sum = 0;
            for (let salary of Object.values(salaries)) {
              sum += salary;
            }
            return sum;
          }


Рішення2:


Для обчислення єдиного значення можна з використати метод масивів .reduce(), але звичайно перед цим потрібно спочатку отримати значення властивостей об'єкту, які потім через метод reduce() додамо та отримаємо результат.
Тобто, для отримання значень ключів об'єтку, використовуємо той самий метод для об'єктів Object.values(obj) (повертає масив значень ключів об'єкту), а вже отримавши ці значення, ітеруємо по них методом масивів reduce(), ось: 


          function sumSalaries(salaries) {
            return Object.values(salaries).reduce((acc, val) => acc + val, 0)
          }
